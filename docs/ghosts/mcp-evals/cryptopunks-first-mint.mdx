## Goal

Find the block hash where the first CryptoPunk NFT was minted. Output must be:

ANSWER: 0x...

## Hints

- Contract address: 0xb47e3cd837dDF8e4c57F05d70Ab865de6e193BBB
- The first mint occurs at contract deployment (look for the deployment transaction and its block).
- Use RPC methods `eth_getCode`, `eth_getTransactionByHash`, and `eth_getBlockByNumber` or an indexer pattern with logs.

## Approach

1. Identify the deployment transaction:
   - Scan logs or check the earliest block where `code` exists (`eth_getCode(address, block)` over a short range).
   - Or search from known deployment block if available.
2. Resolve the transaction’s `blockHash` via `eth_getTransactionByHash`.

## Example (TypeScript + Voltaire JSON-RPC)

```ts
import { createJsonRpcProvider } from '@tevm/voltaire/jsonrpc'

const provider = createJsonRpcProvider(process.env.ETHEREUM_RPC_URL!)
const CRYPTOPUNKS = '0xb47e3cd837dDF8e4c57F05d70Ab865de6e193BBB'

async function deployedAtBlock(address: string): Promise<bigint> {
  let lo = 0n
  let hi = BigInt(await provider.eth_blockNumber())
  while (lo < hi) {
    const mid = (lo + hi) >> 1n
    const code = await provider.eth_getCode(address, `0x${mid.toString(16)}`)
    if (code && code !== '0x') hi = mid
    else lo = mid + 1n
  }
  return lo
}

const blockNumber = await deployedAtBlock(CRYPTOPUNKS)
const block = await provider.eth_getBlockByNumber(`0x${blockNumber.toString(16)}`, false)
console.log(`ANSWER: ${block.hash}`)
```

## Goal

Given token ID and block number, determine owner at that block.

Return as:

ANSWER: 0x...

## Contract

- BAYC: 0xBC4CA0EdA7647A8aB7C2061c2E118A18a936f13D

## Approach

1. Call `ownerOf(tokenId)` via `eth_call` at the historical block.
2. If the contract reverts historically for some IDs, reconstruct via Transfer logs up to that block.

## Example (TypeScript + Voltaire JSON-RPC + ABI)

```ts
import { createJsonRpcProvider } from '@tevm/voltaire/jsonrpc'
import * as Abi from '@tevm/voltaire/Abi'

const provider = createJsonRpcProvider(process.env.ETHEREUM_RPC_URL!)
const BAYC = '0xBC4CA0EdA7647A8aB7C2061c2E118A18a936f13D'

const ownerOfFn = {
  type: 'function',
  name: 'ownerOf',
  inputs: [{ type: 'uint256', name: 'tokenId' }],
  outputs: [{ type: 'address' }],
  stateMutability: 'view',
} as const

const tokenId = 1234n
const blockNumber = 15_000_000n

const selector = Abi.Function.getSelector(ownerOfFn)
const encodedArgs = Abi.Function.encodeParams(ownerOfFn, [tokenId])
const data = `0x${Buffer.from(selector).toString('hex')}${Buffer.from(encodedArgs).toString('hex')}`

const result = await provider.eth_call({ to: BAYC, data }, `0x${blockNumber.toString(16)}`)
const [owner] = Abi.Function.decodeResult(ownerOfFn, result)

console.log(`ANSWER: ${owner}`)
```

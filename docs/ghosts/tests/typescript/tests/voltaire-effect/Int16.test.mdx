## Decode valid positive number

Decodes and validates a positive Int16 value.

```typescript
import * as S from 'effect/Schema'
import * as Int16 from 'voltaire-effect/primitives/Int16'

const value = S.decodeSync(Int16.Number)(1000)
expect(S.encodeSync(Int16.Number)(value)).toBe(1000)
```

## Add two Int16 values

Performs addition with overflow checking.

```typescript
import * as S from 'effect/Schema'
import { BrandedInt16 } from 'voltaire'
import * as Int16 from 'voltaire-effect/primitives/Int16'

const a = S.decodeSync(Int16.Number)(5000)
const b = S.decodeSync(Int16.Number)(3000)
const result = BrandedInt16.plus(a, b)
expect(S.encodeSync(Int16.Number)(result)).toBe(8000)
```

## Check if Int16 is negative

Tests the sign of an Int16 value.

```typescript
import * as S from 'effect/Schema'
import * as Int16 from 'voltaire-effect/primitives/Int16'

const positive = S.decodeSync(Int16.Number)(1000)
expect(Int16.isNegative(positive)).toBe(false)

const negative = S.decodeSync(Int16.Number)(-1000)
expect(Int16.isNegative(negative)).toBe(true)
```

## Compare Int16 values

Returns -1, 0, or 1 for comparison.

```typescript
import * as S from 'effect/Schema'
import * as Int16 from 'voltaire-effect/primitives/Int16'

const a = S.decodeSync(Int16.Number)(100)
const b = S.decodeSync(Int16.Number)(200)
expect(Int16.compare(a, b)).toBe(-1)
```

## Decode hex string

Decodes contract init code from hex string to bytes.

```typescript
import * as S from 'effect/Schema'
import * as InitCode from 'voltaire-effect/primitives/InitCode'

const result = S.decodeSync(InitCode.Hex)('0x608060405234801561001057600080fd5b50')
expect(result).toBeInstanceOf(Uint8Array)
expect(result.length).toBeGreaterThan(0)
```

## Handle empty hex string

Decodes empty init code.

```typescript
import * as S from 'effect/Schema'
import * as InitCode from 'voltaire-effect/primitives/InitCode'

const result = S.decodeSync(InitCode.Hex)('0x')
expect(result.length).toBe(0)
```

## Encode to hex string

Encodes init code bytes back to hex string.

```typescript
import * as S from 'effect/Schema'
import * as InitCode from 'voltaire-effect/primitives/InitCode'

const code = S.decodeSync(InitCode.Hex)('0x6080604052')
const hex = S.encodeSync(InitCode.Hex)(code)
expect(hex).toBe('0x6080604052')
```

## Decode Uint8Array

Decodes init code from a Uint8Array directly.

```typescript
import * as S from 'effect/Schema'
import * as InitCode from 'voltaire-effect/primitives/InitCode'

const input = new Uint8Array([0x60, 0x80, 0x60, 0x40])
const result = S.decodeSync(InitCode.Bytes)(input)
expect([...result]).toEqual([...input])
```

---
title: How to Validate an Ethereum Address
description: Validate address format using Voltaire's Address.isValid function.
---

## Goal

Check if a string or bytes is a valid 20-byte Ethereum address.

## Code

```ts
import { Address } from '@voltaire/primitives'

// Validate hex string
const isValid = Address.isValid("0x742d35Cc6634C0532925a3b844Bc9e7595f251e3")
console.log(isValid) // true

// Invalid cases
Address.isValid("0xinvalid")                    // false - invalid hex
Address.isValid("0x742d35Cc")                   // false - too short
Address.isValid(new Uint8Array(10))             // false - wrong length

// Validate and use
function processAddress(input: string) {
  if (!Address.isValid(input)) {
    throw new Error('Invalid address format')
  }
  const addr = Address.fromHex(input)
  return addr.toHex()
}
```

## Validation Rules

- Must be 40 hex characters (20 bytes)
- With `0x` prefix: 42 characters total
- Valid hex chars only: 0-9, a-f, A-F
- Uint8Array must be exactly 20 bytes

---
title: Check ERC20 Allowance and Approve
description: Check token allowance and approve a spender using Voltaire's Contract module.
---

## Goal

Check how many tokens a spender is allowed to use, then approve a new allowance.

Return allowance as:

ANSWER: `<amount>`

## Example (TypeScript)

```ts
import { Contract } from '@voltaire/contract'

const erc20Abi = [
  {
    type: 'function',
    name: 'allowance',
    stateMutability: 'view',
    inputs: [
      { type: 'address', name: 'owner' },
      { type: 'address', name: 'spender' },
    ],
    outputs: [{ type: 'uint256', name: '' }],
  },
  {
    type: 'function',
    name: 'approve',
    stateMutability: 'nonpayable',
    inputs: [
      { type: 'address', name: 'spender' },
      { type: 'uint256', name: 'amount' },
    ],
    outputs: [{ type: 'bool', name: '' }],
  },
] as const

const usdc = Contract({
  address: '0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48',
  abi: erc20Abi,
  provider,
})

const owner = '0x742d35Cc6634C0532925a3b844Bc454e4438f44e'
const spender = '0x68b3465833fb72A70ecDF485E0e4C7bD8665Fc45' // Uniswap Router

// Check current allowance
const currentAllowance = await usdc.read.allowance(owner, spender)
console.log(`ANSWER: ${currentAllowance}`)

// Approve new allowance
const newAllowance = 1000000000n // 1000 USDC
const txHash = await usdc.write.approve(spender, newAllowance)
```

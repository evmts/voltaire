---
title: Wrap and Unwrap ETH (WETH)
description: Convert between ETH and WETH using Voltaire's Contract module.
---

## Goal

Wrap ETH to WETH or unwrap WETH back to ETH.

Return transaction hash as:

ANSWER: 0x...

## Example (TypeScript)

```ts
import { Contract } from '@voltaire/contract'
import * as Hex from '@voltaire/primitives/Hex'

const wethAbi = [
  {
    type: 'function',
    name: 'deposit',
    stateMutability: 'payable',
    inputs: [],
    outputs: [],
  },
  {
    type: 'function',
    name: 'withdraw',
    stateMutability: 'nonpayable',
    inputs: [{ type: 'uint256', name: 'wad' }],
    outputs: [],
  },
  {
    type: 'function',
    name: 'balanceOf',
    stateMutability: 'view',
    inputs: [{ type: 'address', name: '' }],
    outputs: [{ type: 'uint256', name: '' }],
  },
] as const

const weth = Contract({
  address: '0xC02aaA39b223FE8D0A0e5C4F27eAD9083C756Cc2', // Mainnet WETH
  abi: wethAbi,
  provider,
})

// Wrap ETH -> WETH (send ETH with deposit call)
const wrapAmount = 1000000000000000000n // 1 ETH
const addressHex = Hex.fromBytes(weth.address)
const depositTx = await provider.request({
  method: 'eth_sendTransaction',
  params: [{
    to: addressHex,
    value: `0x${wrapAmount.toString(16)}`,
    data: '0xd0e30db0', // deposit() selector
  }],
})
console.log(`Wrap ANSWER: ${depositTx}`)

// Unwrap WETH -> ETH
const unwrapAmount = 500000000000000000n // 0.5 WETH
const txHash = await weth.write.withdraw(unwrapAmount)
console.log(`Unwrap ANSWER: 0x${Buffer.from(txHash).toString('hex')}`)

// Check WETH balance
const account = '0x742d35Cc6634C0532925a3b844Bc454e4438f44e'
const balance = await weth.read.balanceOf(account)
console.log(`WETH Balance: ${balance}`)
```

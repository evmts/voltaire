---
title: Transfer ERC20 Tokens
description: Send ERC20 tokens to another address using Voltaire's Contract module.
---

## Goal

Transfer ERC20 tokens from sender to recipient.

Return transaction hash as:

ANSWER: 0x...

## Example (TypeScript)

```ts
import { Contract } from '@voltaire/contract'
import * as S from 'effect/Schema'
import * as AbiSchema from 'voltaire-effect/primitives/Abi'

const erc20Abi = S.decodeUnknownSync(AbiSchema.fromArray)([
  {
    type: 'function',
    name: 'transfer',
    stateMutability: 'nonpayable',
    inputs: [
      { type: 'address', name: 'to' },
      { type: 'uint256', name: 'amount' },
    ],
    outputs: [{ type: 'bool', name: '' }],
  },
])

const usdc = Contract({
  address: '0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48',
  abi: erc20Abi,
  provider,
})

const recipient = '0x742d35Cc6634C0532925a3b844Bc454e4438f44e'
const amount = 1000000n // 1 USDC (6 decimals)

const txHash = await usdc.write.transfer(recipient, amount)
console.log(`ANSWER: 0x${Buffer.from(txHash).toString('hex')}`)
```

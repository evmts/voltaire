## Using eth_sendRawTransaction

```typescript
import * as Transaction from 'voltaire/primitives/Transaction';
import * as Hex from 'voltaire/primitives/Hex';
import { HttpProvider } from 'voltaire/provider';

const provider = new HttpProvider('https://eth.llamarpc.com');

// Serialize signed transaction
const serialized = Transaction.serialize(signedTx);
const rawTxHex = Hex.fromBytes(serialized);

// Broadcast to network
const txHash = await provider.request({
  method: 'eth_sendRawTransaction',
  params: [rawTxHex]
});

console.log('Transaction hash:', txHash);
// 0x1234...abcd
```

## Full Example

```typescript
import * as Transaction from 'voltaire/primitives/Transaction';
import * as Address from 'voltaire/primitives/Address';
import * as Hex from 'voltaire/primitives/Hex';
import { PrivateKeySignerImpl } from 'voltaire/crypto/signers/private-key-signer';
import { HttpProvider } from 'voltaire/provider';

const provider = new HttpProvider('https://eth.llamarpc.com');
const signer = PrivateKeySignerImpl.fromPrivateKey({
  privateKey: process.env.PRIVATE_KEY
});

// Get nonce
const nonce = await provider.request({
  method: 'eth_getTransactionCount',
  params: [signer.address, 'pending']
});

// Build transaction
const tx: Transaction.EIP1559 = {
  type: 2,
  chainId: 1n,
  nonce: BigInt(nonce),
  maxFeePerGas: 30000000000n,
  maxPriorityFeePerGas: 2000000000n,
  gasLimit: 21000n,
  to: Address.from('0xRecipient'),
  value: 1000000000000000000n,
  data: new Uint8Array(),
  accessList: []
};

// Sign
const signedTx = await signer.signTransaction(tx);

// Broadcast
const txHash = await provider.request({
  method: 'eth_sendRawTransaction',
  params: [Hex.fromBytes(Transaction.serialize(signedTx))]
});

console.log('Submitted:', txHash);
```

## Error Handling

```typescript
try {
  const txHash = await provider.request({
    method: 'eth_sendRawTransaction',
    params: [rawTxHex]
  });
} catch (error) {
  if (error.message.includes('nonce too low')) {
    console.log('Nonce already used - fetch fresh nonce');
  } else if (error.message.includes('insufficient funds')) {
    console.log('Not enough ETH for value + gas');
  } else if (error.message.includes('replacement transaction underpriced')) {
    console.log('Bump gas price by at least 10%');
  }
}
```

## Key Points

- `eth_sendRawTransaction` takes hex-encoded signed tx
- Returns transaction hash immediately (does NOT mean confirmed)
- Common errors: nonce issues, insufficient funds, underpriced replacement
- Use `eth_getTransactionReceipt` to check confirmation

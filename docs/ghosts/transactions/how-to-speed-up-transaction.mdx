---
title: How to Speed Up a Transaction
description: Speed up a pending transaction by resubmitting with higher gas
---

## Using Transaction.replaceWith

```typescript
import * as Transaction from 'voltaire/primitives/Transaction';

// Original pending transaction
const pendingTx: Transaction.EIP1559 = {
  type: 2,
  chainId: 1n,
  nonce: 5n, // Must use SAME nonce
  maxFeePerGas: 20000000000n,
  maxPriorityFeePerGas: 1000000000n,
  gasLimit: 21000n,
  to: recipientAddress,
  value: 1000000000000000000n,
  data: new Uint8Array(),
  accessList: []
};

// Create replacement with 10% higher fees (default)
const speedUpTx = Transaction.replaceWith(pendingTx);

// Or specify custom bump percentage
const speedUpTx20 = Transaction.replaceWith(pendingTx, { bumpPercentage: 20 });

// Or specify exact new fees
const speedUpTxExact = Transaction.replaceWith(pendingTx, {
  maxFeePerGas: 30000000000n,
  maxPriorityFeePerGas: 2000000000n
});
```

## Manual Speed-Up

```typescript
// Same nonce, higher fees
const speedUpTx: Transaction.EIP1559 = {
  ...pendingTx,
  maxFeePerGas: pendingTx.maxFeePerGas * 110n / 100n, // +10%
  maxPriorityFeePerGas: pendingTx.maxPriorityFeePerGas * 110n / 100n
};

// Sign and broadcast
const signedTx = await signer.signTransaction(speedUpTx);
const serialized = Transaction.serialize(signedTx);
const txHash = await provider.request({
  method: 'eth_sendRawTransaction',
  params: [`0x${Buffer.from(serialized).toString('hex')}`]
});
```

## Legacy Transaction Speed-Up

```typescript
const speedUpLegacy = Transaction.replaceWith(pendingLegacyTx, {
  gasPrice: pendingLegacyTx.gasPrice * 2n // Double gas price
});
```

## Key Points

- **Same nonce required** - replacement tx must use identical nonce
- **Minimum 10% bump** - most nodes require at least 10% fee increase
- **Sign again** - replacement needs fresh signature
- **Only one confirms** - original or replacement, whichever miner picks first

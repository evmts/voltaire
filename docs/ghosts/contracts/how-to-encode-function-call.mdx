---
title: How to Encode Function Call Data
description: ABI-encode function call data for smart contract interactions
---

## Goal

Encode function call data (4-byte selector + encoded params) for `eth_call` or transaction data.

## Using Contract

```typescript
import { Contract } from '@voltaire/contract';

const abi = [
  {
    type: 'function',
    name: 'transfer',
    inputs: [
      { type: 'address', name: 'to' },
      { type: 'uint256', name: 'amount' }
    ],
    outputs: [{ type: 'bool' }]
  }
] as const;

const contract = Contract({ address: '0x...', abi, provider });

// Encode using abi.encode()
const calldata = contract.abi.encode('transfer', [
  '0x742d35Cc6634C0532925a3b844Bc9e7595f251e3',
  1000n
]);
// Returns Uint8Array: 0xa9059cbb...
```

## Using Abi Directly

```typescript
import * as Abi from '@voltaire/primitives/Abi';

const abi = Abi.from([
  {
    type: 'function',
    name: 'transfer',
    inputs: [
      { type: 'address', name: 'to' },
      { type: 'uint256', name: 'amount' }
    ]
  }
]);

// Encode function call
const encoded = Abi.encodeFunction(abi, 'transfer', [
  '0x742d35Cc6634C0532925a3b844Bc9e7595f251e3',
  1000n
]);
// Returns HexType: 0xa9059cbb000000000000000000000000742d35cc6634c0532925a3b844bc9e7595f251e30000000000000000000000000000000000000000000000000000000000003e8
```

## Low-Level Function Encoding

```typescript
import * as Abi from '@voltaire/primitives/Abi';

const transferFn = {
  type: 'function',
  name: 'transfer',
  inputs: [
    { type: 'address', name: 'to' },
    { type: 'uint256', name: 'amount' }
  ]
} as const;

// Get 4-byte selector
const selector = Abi.Function.getSelector(transferFn);
// 0xa9059cbb

// Encode params (includes selector)
const calldata = Abi.Function.encodeParams(transferFn, [
  '0x742d35Cc6634C0532925a3b844Bc9e7595f251e3',
  1000n
]);
```

  <p>Guillotine is built from the ground up using data-oriented design principles:</p>
  
  <ul style={{ lineHeight: '1.8' }}>
    <li>**Optimized Stack Frame** - Cache-conscious struct layout for minimal memory access</li>
    <li>**Indirect Threading** - Tail-call recursion for excellent CPU branch prediction</li>
    <li>**Opcode Fusion** - Common patterns compiled into single dispatch operations</li>
    <li>**Assembly Keccak** - Hardware-accelerated cryptography via keccak-asm</li>
    <li>**Batched Gas Calculation** - Static costs computed in advance</li>
  </ul>

## ğŸš€ Quick Start

Get up and running with Guillotine in seconds:

::::steps
### Install Zig

```bash
# macOS
brew install zig

# Linux
wget https://ziglang.org/download/0.14.1/zig-linux-x86_64-0.14.1.tar.xz
tar -xf zig-linux-x86_64-0.14.1.tar.xz

# Windows
scoop install zig
```

### Clone & Build

```bash
git clone https://github.com/evmts/Guillotine.git
cd Guillotine
zig build
```

### Run Tests

```bash
zig build test
```

### Use in Your Project

```zig
const guillotine = @import("guillotine");

pub fn main() !void {
    var evm = try guillotine.Evm.init(allocator);
    defer evm.deinit();
    
    const result = try evm.execute(bytecode);
}
```

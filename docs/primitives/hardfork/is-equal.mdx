## Alias Handling

`isEqual` correctly handles hardfork aliases:

```typescript
import { Hardfork } from '@tevm/voltaire'

const merge = Hardfork.fromString("merge")
const paris = Hardfork.fromString("paris")

// Both refer to same hardfork
merge.isEqual(paris)  // true
```

### Supported Aliases

| Canonical Name | Alias | Reason |
|---------------|-------|--------|
| merge | paris | Consensus layer name |
| petersburg | constantinoplefix | Alternative name |

## Usage Patterns

### Configuration Validation

Compare user input against expected version:

```typescript
import { Hardfork, CANCUN } from '@tevm/voltaire'

function validateConfig(config: Config) {
  const fork = Hardfork.fromString(config.hardfork)

  if (!fork.isEqual(CANCUN)) {
    throw new Error(`Expected Cancun, got ${Hardfork.toString(fork)}`)
  }
}
```

### Network Configuration

Check if networks use same hardfork:

```typescript
import { Hardfork } from '@tevm/voltaire'

function networksCompatible(net1: Network, net2: Network): boolean {
  const fork1 = Hardfork.fromString(net1.hardfork)
  const fork2 = Hardfork.fromString(net2.hardfork)

  return fork1.isEqual(fork2)
}
```

### Conditional Logic

Branch based on specific hardfork:

```typescript
import { Hardfork, MERGE } from '@tevm/voltaire'

function getConsensusInfo(fork: BrandedHardfork) {
  if (fork.isEqual(MERGE)) {
    return {
      name: "The Merge",
      consensus: "Proof of Stake",
      date: "September 15, 2022"
    }
  }
  // ...
}
```

## String Comparison Caveat

**Don't** compare hardfork strings directly with `===`:

```typescript
import { Hardfork } from '@tevm/voltaire'

const merge = Hardfork.fromString("merge")
const paris = Hardfork.fromString("paris")

// Bad - fails with aliases
merge === paris  // false (different strings)

// Good - handles aliases
merge.isEqual(paris)  // true (same hardfork)
```

## Normalization

For storing or comparing, normalize to canonical name:

```typescript
import { Hardfork } from '@tevm/voltaire'

function normalizeHardfork(input: string): string {
  const fork = Hardfork.fromString(input)
  return Hardfork.toString(fork)
}

normalizeHardfork("merge")  // "merge"
normalizeHardfork("paris")  // "merge" (canonical)
normalizeHardfork("Merge")  // "merge" (lowercase)
```

## Usage Examples

### Encode Different Types

```typescript
import * as Parameter from '@tevm/voltaire/Abi/parameter'

// Address
const addrParam = { type: "address", name: "to" }
const encodedAddr = Parameter.encode(addrParam, "0x742d35Cc6634C0532925a3b844Bc9e7595f51e3e")

// Uint256
const uint256Param = { type: "uint256", name: "amount" }
const encodedUint = Parameter.encode(uint256Param, 1000n)

// String
const stringParam = { type: "string", name: "name" }
const encodedString = Parameter.encode(stringParam, "Alice")

// Bool
const boolParam = { type: "bool", name: "active" }
const encodedBool = Parameter.encode(boolParam, true)
```

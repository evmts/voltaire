## Usage Examples

### Build Function Definition

```typescript
import * as Parameter from '@tevm/voltaire/Abi/parameter'

const transferFn = {
  type: "function",
  name: "transfer",
  inputs: [
    Parameter("address", "to"),
    Parameter("uint256", "amount")
  ],
  outputs: [
    Parameter("bool", "success")
  ]
}
```

### Complex Types

```typescript
import * as Parameter from '@tevm/voltaire/Abi/parameter'

// Array types
const addressesParam = Parameter("address[]", "recipients")
const fixedArrayParam = Parameter("uint256[3]", "values")

// Bytes types
const bytesParam = Parameter("bytes", "data")
const bytes32Param = Parameter("bytes32", "hash")
```

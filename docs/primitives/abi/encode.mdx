## `encode(item: AbiItem, args: unknown[]): Hex`

Generic encoding for any ABI item type (function, error, constructor).

**Parameters:**
- `item: AbiItem` - ABI item to encode for
- `args: unknown[]` - Argument values

**Returns:** `Hex` - Encoded data with selector (for function/error) or without (for constructor)

**Example:**

```typescript
import * as BrandedAbi from '@tevm/voltaire/Abi/BrandedAbi'

const transferFn = {
  type: "function",
  name: "transfer",
  inputs: [
    { type: "address", name: "to" },
    { type: "uint256", name: "amount" }
  ]
}

const encoded = BrandedAbi.encode(transferFn, [
  "0x742d35Cc6634C0532925a3b844Bc9e7595f51e3e",
  1000n
])
```

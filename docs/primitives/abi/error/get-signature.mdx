## `error.getSignature(): string`

Get the canonical error signature string in the format `ErrorName(type1,type2,...)`.

**Returns:** `string` - Canonical error signature

**Example:**

```typescript
import { AbiError } from '@tevm/voltaire'

const insufficientBalance = new AbiError({
  type: "error",
  name: "InsufficientBalance",
  inputs: [
    { type: "uint256", name: "balance" },
    { type: "uint256", name: "required" }
  ]
})

const signature = insufficientBalance.getSignature()
console.log(signature)  // "InsufficientBalance(uint256,uint256)"
```

## `getSignature(error: BrandedError): string`

Get the canonical error signature string.

**Parameters:**
- `error: BrandedError` - Error ABI item

**Returns:** `string` - Canonical error signature

**Example:**

```typescript
import * as AbiError from '@tevm/voltaire/Abi/error'

const insufficientBalance = {
  type: "error",
  name: "InsufficientBalance",
  inputs: [
    { type: "uint256", name: "balance" },
    { type: "uint256", name: "required" }
  ]
}

const signature = AbiError.getSignature(insufficientBalance)
```

**Defined in:** [primitives/Abi/error/BrandedError/getSignature.js](https://github.com/evmts/voltaire/blob/main/src/primitives/Abi/error/BrandedError/getSignature.js)

## `getItem(abi: Abi, name: string, type: string): AbiItem | undefined`

Find a specific ABI item by name and type from an ABI array.

**Parameters:**
- `abi: Abi` - ABI array
- `name: string` - Item name
- `type: string` - Item type ("function", "event", "error", "constructor")

**Returns:** `AbiItem | undefined` - Found item or undefined

**Example:**

```typescript
import * as BrandedAbi from '@tevm/voltaire/Abi/BrandedAbi'

const abi = [
  {
    type: "function",
    name: "transfer",
    inputs: [
      { type: "address", name: "to" },
      { type: "uint256", name: "amount" }
    ]
  },
  {
    type: "event",
    name: "Transfer",
    inputs: [
      { type: "address", name: "from", indexed: true },
      { type: "address", name: "to", indexed: true },
      { type: "uint256", name: "value" }
    ]
  }
]

const transferFn = BrandedAbi.getItem(abi, "transfer", "function")
const transferEvent = BrandedAbi.getItem(abi, "Transfer", "event")
```

## Usage Examples

### Find and Use Function

```typescript
import * as BrandedAbi from '@tevm/voltaire/Abi/BrandedAbi'

const abi = [...] // Full contract ABI

const transferFn = BrandedAbi.getItem(abi, "transfer", "function")

if (transferFn && transferFn.type === "function") {
  const calldata = BrandedAbi.Function.encodeParams(transferFn, [to, amount])
  await provider.call({ to: contractAddress, data: calldata })
}
```

### Error Handling

```typescript
import * as BrandedAbi from '@tevm/voltaire/Abi/BrandedAbi'

const abi = [...]

const item = BrandedAbi.getItem(abi, "nonExistent", "function")
if (!item) {
  throw new Error("Function not found in ABI")
}
```

## `parseLogs(abi: Abi, logs: Log[]): DecodedLog[]`

Parse an array of logs using the full contract ABI, automatically matching events by selector.

**Parameters:**
- `abi: Abi` - Full contract ABI with event definitions
- `logs: Log[]` - Array of raw logs from blockchain

**Returns:** `DecodedLog[]` - Array of decoded logs with event data

**Example:**

```typescript
import * as BrandedAbi from '@tevm/voltaire/Abi/BrandedAbi'

const abi = [
  {
    type: "event",
    name: "Transfer",
    inputs: [
      { type: "address", name: "from", indexed: true },
      { type: "address", name: "to", indexed: true },
      { type: "uint256", name: "value" }
    ]
  },
  {
    type: "event",
    name: "Approval",
    inputs: [
      { type: "address", name: "owner", indexed: true },
      { type: "address", name: "spender", indexed: true },
      { type: "uint256", name: "value" }
    ]
  }
]

const logs = await provider.getLogs({
  address: tokenAddress,
  fromBlock: 'latest'
})

const decoded = BrandedAbi.parseLogs(abi, logs)

for (const log of decoded) {
  console.log(`Event: ${log.eventName}`)
  console.log("Args:", log.args)
}
```

**Defined in:** [primitives/Abi/BrandedAbi/parseLogs.js](https://github.com/evmts/voltaire/blob/main/src/primitives/Abi/BrandedAbi/parseLogs.js)

## Usage Examples

### Parse All Contract Events

```typescript
import * as BrandedAbi from '@tevm/voltaire/Abi/BrandedAbi'

const abi = [...] // Full ERC20 ABI

const logs = await provider.getLogs({
  address: tokenAddress,
  fromBlock: startBlock,
  toBlock: endBlock
})

const decoded = BrandedAbi.parseLogs(abi, logs)

// Process decoded events
for (const log of decoded) {
  if (log.eventName === "Transfer") {
    console.log(`Transfer: ${log.args.value} from ${log.args.from} to ${log.args.to}`)
  } else if (log.eventName === "Approval") {
    console.log(`Approval: ${log.args.value} from ${log.args.owner} to ${log.args.spender}`)
  }
}
```

### Filter Specific Event Type

```typescript
import * as BrandedAbi from '@tevm/voltaire/Abi/BrandedAbi'

const abi = [...]
const logs = await provider.getLogs({ address: contractAddress })

const decoded = BrandedAbi.parseLogs(abi, logs)

// Get only Transfer events
const transfers = decoded.filter(log => log.eventName === "Transfer")

for (const transfer of transfers) {
  console.log(transfer.args)
}
```

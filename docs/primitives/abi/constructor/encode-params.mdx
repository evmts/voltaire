## `constructor.encodeParams(args: unknown[]): Hex`

Encode constructor parameters to append to contract bytecode during deployment.

**Parameters:**
- `args: unknown[]` - Array of constructor parameter values

**Returns:** `Hex` - ABI-encoded constructor parameters

**Example:**

```typescript
import { Constructor } from '@tevm/voltaire'

const constructorDef = new Constructor({
  type: "constructor",
  inputs: [
    { type: "string", name: "name" },
    { type: "string", name: "symbol" },
    { type: "uint8", name: "decimals" }
  ]
})

const encoded = constructorDef.encodeParams(["MyToken", "MTK", 18])

// Append to bytecode for deployment
const deployData = contractBytecode + encoded.slice(2)  // Remove 0x
```

## `encodeParams(constructor: BrandedConstructor, args: unknown[]): Hex`

Encode constructor parameters.

**Parameters:**
- `constructor: BrandedConstructor` - Constructor ABI item
- `args: unknown[]` - Parameter values

**Returns:** `Hex` - Encoded constructor parameters

**Example:**

```typescript
import * as Constructor from '@tevm/voltaire/Abi/constructor'

const constructorDef = {
  type: "constructor",
  inputs: [
    { type: "string", name: "name" },
    { type: "string", name: "symbol" },
    { type: "uint8", name: "decimals" }
  ]
}

const encoded = Constructor.encodeParams(constructorDef, ["MyToken", "MTK", 18])
```

**Defined in:** [primitives/Abi/constructor/BrandedConstructor/encodeParams.js](https://github.com/evmts/voltaire/blob/main/src/primitives/Abi/constructor/BrandedConstructor/encodeParams.js)

## Usage Examples

### Deploy ERC20 Contract

```typescript
import { Constructor } from '@tevm/voltaire'

const constructorDef = new Constructor({
  type: "constructor",
  inputs: [
    { type: "string", name: "name" },
    { type: "string", name: "symbol" },
    { type: "uint8", name: "decimals" }
  ]
})

const encoded = constructorDef.encodeParams(["MyToken", "MTK", 18])

// Create deployment transaction
const tx = {
  data: contractBytecode + encoded.slice(2),
  value: 0n
}

const receipt = await wallet.sendTransaction(tx)
console.log("Contract deployed at:", receipt.contractAddress)
```

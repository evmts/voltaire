## `decode(item: AbiItem, data: Hex): unknown[]`

Generic decoding for any ABI item type.

**Parameters:**
- `item: AbiItem` - ABI item to decode for
- `data: Hex` - Encoded data to decode

**Returns:** `unknown[]` - Decoded parameter values

**Example:**

```typescript
import * as BrandedAbi from '@tevm/voltaire/Abi/BrandedAbi'

const transferFn = {
  type: "function",
  name: "transfer",
  inputs: [
    { type: "address", name: "to" },
    { type: "uint256", name: "amount" }
  ]
}

const calldata = "0xa9059cbb..."

const [to, amount] = BrandedAbi.decode(transferFn, calldata)
```

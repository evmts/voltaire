## `event.getSelector(): Hex`

Get the event signature hash by hashing the event signature with keccak256. This becomes topic 0 in event logs.

**Returns:** `Hex` - 32-byte event signature hash

**Example:**

```typescript
import { Event } from '@tevm/voltaire'

const transferEvent = new Event({
  type: "event",
  name: "Transfer",
  inputs: [
    { type: "address", name: "from", indexed: true },
    { type: "address", name: "to", indexed: true },
    { type: "uint256", name: "value", indexed: false }
  ]
})

const selector = transferEvent.getSelector()
console.log(selector)
// "0xddf252ad1be2c89b69c2b068fc378daa952ba7f163c4a11628f55a4df523b3ef"
```

## `getSelector(event: BrandedEvent): Hex`

Get the event signature hash by hashing the event signature with keccak256.

**Parameters:**
- `event: BrandedEvent` - Event ABI item

**Returns:** `Hex` - 32-byte event signature hash

**Example:**

```typescript
import * as Event from '@tevm/voltaire/Abi/event'

const transferEvent = {
  type: "event",
  name: "Transfer",
  inputs: [
    { type: "address", name: "from", indexed: true },
    { type: "address", name: "to", indexed: true },
    { type: "uint256", name: "value", indexed: false }
  ]
}

const selector = Event.getSelector(transferEvent)
```

**Defined in:** [primitives/Abi/event/BrandedEvent/getSelector.js](https://github.com/evmts/voltaire/blob/main/src/primitives/Abi/event/BrandedEvent/getSelector.js)

## Usage Examples

### Event Filtering

```typescript
import { Event } from '@tevm/voltaire'

const transferEvent = new Event({
  type: "event",
  name: "Transfer",
  inputs: [
    { type: "address", name: "from", indexed: true },
    { type: "address", name: "to", indexed: true },
    { type: "uint256", name: "value", indexed: false }
  ]
})

// Get logs for Transfer events
const logs = await provider.getLogs({
  address: tokenAddress,
  topics: [transferEvent.getSelector()],
  fromBlock: 'latest'
})
```

### Multiple Event Types

```typescript
import { Event } from '@tevm/voltaire'

const transferEvent = new Event({
  type: "event",
  name: "Transfer",
  inputs: [
    { type: "address", name: "from", indexed: true },
    { type: "address", name: "to", indexed: true },
    { type: "uint256", name: "value" }
  ]
})

const approvalEvent = new Event({
  type: "event",
  name: "Approval",
  inputs: [
    { type: "address", name: "owner", indexed: true },
    { type: "address", name: "spender", indexed: true },
    { type: "uint256", name: "value" }
  ]
})

// Filter for either event type
const logs = await provider.getLogs({
  address: tokenAddress,
  topics: [[
    transferEvent.getSelector(),
    approvalEvent.getSelector()
  ]]
})
```

### Event Registry

```typescript
import { Event } from '@tevm/voltaire'

const events = [
  new Event({
    type: "event",
    name: "Transfer",
    inputs: [
      { type: "address", name: "from", indexed: true },
      { type: "address", name: "to", indexed: true },
      { type: "uint256", name: "value" }
    ]
  }),
  new Event({
    type: "event",
    name: "Approval",
    inputs: [
      { type: "address", name: "owner", indexed: true },
      { type: "address", name: "spender", indexed: true },
      { type: "uint256", name: "value" }
    ]
  })
]

// Create selector â†’ event mapping
const registry = new Map(
  events.map(e => [e.getSelector(), e])
)

// Look up event by selector
function findEvent(log) {
  const selector = log.topics[0]
  return registry.get(selector)
}
```

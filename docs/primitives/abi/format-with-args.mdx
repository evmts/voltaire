## `formatWithArgs(item: AbiItem, args: unknown[]): string`

Format an ABI item with actual argument values for debugging and logging.

**Parameters:**
- `item: AbiItem` - Any ABI item
- `args: unknown[]` - Argument values to display

**Returns:** `string` - Formatted string with values

**Example:**

```typescript
import * as BrandedAbi from '@tevm/voltaire/Abi/BrandedAbi'

const transferFn = {
  type: "function",
  name: "transfer",
  inputs: [
    { type: "address", name: "to" },
    { type: "uint256", name: "amount" }
  ]
}

const formatted = BrandedAbi.formatWithArgs(transferFn, [
  "0x742d35Cc6634C0532925a3b844Bc9e7595f51e3e",
  1000n
])

console.log(formatted)
// "transfer(0x742d35Cc6634C0532925a3b844Bc9e7595f51e3e, 1000)"
```

## Usage Examples

### Debug Transaction

```typescript
import * as BrandedAbi from '@tevm/voltaire/Abi/BrandedAbi'

const transferFn = {
  type: "function",
  name: "transfer",
  inputs: [
    { type: "address", name: "to" },
    { type: "uint256", name: "amount" }
  ]
}

const to = "0x742d35Cc6634C0532925a3b844Bc9e7595f51e3e"
const amount = 1000n

console.log("Calling:", BrandedAbi.formatWithArgs(transferFn, [to, amount]))
// "Calling: transfer(0x742d35Cc6634C0532925a3b844Bc9e7595f51e3e, 1000)"
```

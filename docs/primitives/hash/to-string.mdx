## `hash.toString(): string`

Converts hash to hex string. Alias for `toHex()`.

**Returns:** `string` - Lowercase hex string with `0x` prefix (66 chars)

**Example:**

```typescript
import { Hash } from '@tevm/voltaire'

const hash = Hash.keccak256String("hello")
hash.toString()
// "0x1c8aff950685c2ed4bc3174f3472287b56d9517b9c948127319a09a7a36deac8"

// Useful for string interpolation
console.log(`Hash: ${hash}`)
// Hash: 0x1c8aff950685c2ed4bc3174f3472287b56d9517b9c948127319a09a7a36deac8

// Equivalent to toHex()
hash.toString() === hash.toHex() // true
```

**Static form:**
```typescript
Hash.toString(hash) // "0x..."
```

## Alias for toHex

`toString()` is a direct alias for `toHex()` - they produce identical output:

```typescript
const hash = Hash.keccak256String("hello")

hash.toString()
// "0x1c8aff950685c2ed4bc3174f3472287b56d9517b9c948127319a09a7a36deac8"

hash.toHex()
// "0x1c8aff950685c2ed4bc3174f3472287b56d9517b9c948127319a09a7a36deac8"

// Identical output
hash.toString() === hash.toHex() // true
```

## Automatic String Conversion

JavaScript automatically calls `toString()` in string contexts:

```typescript

// String interpolation
console.log(`Hash value: ${hash}`)
// Hash value: 0x1c8aff950685c2ed4bc3174f3472287b56d9517b9c948127319a09a7a36deac8

// String concatenation
const message = "Transaction hash: " + hash
// Transaction hash: 0x1c8aff950685c2ed4bc3174f3472287b56d9517b9c948127319a09a7a36deac8

// Template literals
```

## When to Use toString vs toHex

Both methods are identical - use whichever is more semantically clear:

**Use `toString()`:**
- String interpolation/concatenation
- Logging and debugging
- Human-readable output
- When JavaScript calls it automatically

**Use `toHex()`:**
- Explicit hex conversion
- API responses (clarity about format)
- When hex format matters semantically

```typescript
// Logging - toString is natural
console.log(`Hash: ${hash}`)

// API response - toHex is clearer
return {
  txHash: hash.toHex()
}

// Debug output - toString is natural
throw new Error(`Invalid hash: ${hash}`)

// Hex formatting - toHex is clearer
const hexString = hash.toHex()
```

## Performance

Identical performance to `toHex()` - they're the same function:

```typescript
// Same implementation
hash.toString() === hash.toHex() // true

// Same performance
console.time('toString')
for (let i = 0; i < 10000; i++) {
  hash.toString()
}
console.timeEnd('toString') // ~500μs

console.time('toHex')
for (let i = 0; i < 10000; i++) {
  hash.toHex()
}
console.timeEnd('toHex') // ~500μs
```

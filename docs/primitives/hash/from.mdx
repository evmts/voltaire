## HashLike Type

`HashLike` is a union type accepting any input that can be coerced to a hash:

```typescript
type HashLike =
  | Uint8Array
  | BrandedHash
  | Hash
  | Hex
  | string
```

### Accepted Types

**Uint8Array** - Raw byte arrays (must be 32 bytes):
```typescript
const bytes = new Uint8Array(32)
const hash = Hash(bytes)
```

**BrandedHash** - Branded hash (passed through):
```typescript
const branded = Hash("0x1c8aff...")
const hash = Hash(branded)  // No conversion
```

**Hex String** - With or without `0x` prefix (must be 64 hex chars):
```typescript
const hash1 = Hash("0x1c8aff950685c2ed4bc3174f3472287b56d9517b9c948127319a09a7a36deac8")
const hash2 = Hash("1c8aff950685c2ed4bc3174f3472287b56d9517b9c948127319a09a7a36deac8")
```

<Warning title="String Interpretation">
Strings without `0x` prefix must be exactly 64 hex characters. Use `0x` prefix for clarity.
</Warning>

### Usage in Function Signatures

Use `HashLike` for flexible function parameters:

```typescript
function verifyHash(input: HashLike): boolean {
  const hash = Hash(input)
  return hash.isZero()
}

// All valid calls
verifyHash("0x1c8aff950685c2ed4bc3174f3472287b56d9517b9c948127319a09a7a36deac8")
verifyHash(new Uint8Array(32))
verifyHash(Hash("0x1c8aff..."))
```

### Performance

**No conversion overhead** for `Uint8Array`, `BrandedHash`, and `Hash` (zero-copy).

**Conversion required** for strings (parses hex and allocates new Uint8Array).

For performance-critical code, prefer passing `Uint8Array` or `BrandedHash` directly.

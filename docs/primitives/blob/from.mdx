## Signature

```typescript
Blob(value: Uint8Array): Blob
```

Create blob from Uint8Array. If exactly 131,072 bytes, treats as raw blob. Otherwise, encodes using `fromData()` format (length prefix + data + padding).

**Alias:** `Blob.from()` (deprecated, use `Blob()` instead)

### Parameters

- **value** `Uint8Array` - Either raw blob (131,072 bytes) or data to encode

### Returns

`Blob` - Blob instance (always exactly 131,072 bytes)

### Throws

- If encoded data exceeds maximum size (~131,064 bytes)

### Example

```typescript
import { Blob } from '@tevm/voltaire';

// Raw blob (exactly 131,072 bytes)
const rawBlob = new Uint8Array(131072);
const blob1 = Blob(rawBlob);

// Data to encode (auto-detects and encodes)
const data = new TextEncoder().encode("Hello");
const blob2 = Blob(data); // Encoded with length prefix

console.log(blob1.length); // 131072
console.log(blob2.length); // 131072
```

## Signature

```zig
pub fn from(allocator: std.mem.Allocator, value: []const u8) !Blob
```

Create blob from bytes. Auto-detects raw blob vs data to encode.

### Parameters

- **allocator** `std.mem.Allocator` - Memory allocator
- **value** `[]const u8` - Either raw blob or data to encode

### Returns

`Blob` - Allocated blob (caller owns memory)

### Errors

- `error.DataTooLarge` - Data exceeds maximum size

### Example

```zig
const Blob = @import("primitives").Blob;

// Raw blob
var raw_blob: [131072]u8 = undefined;
const blob1 = try Blob.from(allocator, &raw_blob);
defer blob1.deinit(allocator);

// Data to encode
const data = "Hello";
const blob2 = try Blob.from(allocator, data);
defer blob2.deinit(allocator);
```

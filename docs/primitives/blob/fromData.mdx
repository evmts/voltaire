## Signature

```typescript
Blob.fromData(data: Uint8Array): Blob
```

Encode arbitrary data into standard blob format with length prefix. Creates 131,072-byte blob containing: 8-byte length prefix (little-endian uint64) + data + zero padding.

### Parameters

- **data** `Uint8Array` - Data to encode (max ~131,064 bytes)

### Returns

`Blob` - Blob containing encoded data (always exactly 131,072 bytes)

### Throws

- If data exceeds maximum size (131,064 bytes = SIZE - 8)

### Example

```typescript
import { Blob } from '@tevm/voltaire';

// Encode text
const text = new TextEncoder().encode("L2 rollup batch #42");
const blob = Blob.fromData(text);

console.log(blob.length); // 131,072

// Later: extract data
const decoded = Blob.toData(blob);
console.log(new TextDecoder().decode(decoded)); // "L2 rollup batch #42"
```

### Encoding Format

```
Bytes 0-7:     Data length (8-byte little-endian uint64)
Bytes 8-N:     Actual data
Bytes N+1-131071: Zero padding
```

### Example with Large Data

```typescript
import { Blob } from '@tevm/voltaire';

// Maximum data size
const maxData = new Uint8Array(131072 - 8); // 131,064 bytes
const blob = Blob.fromData(maxData); // OK

// Too large
try {
  const tooLarge = new Uint8Array(131072);
  Blob.fromData(tooLarge); // Error
} catch (e) {
  console.error(e); // "Data too large: 131072 bytes (max 131064)"
}
```

<Tab title="Zig">

## Signature

```zig
pub fn fromData(allocator: std.mem.Allocator, data: []const u8) !Blob
```

Encode data into blob format with length prefix.

### Parameters

- **allocator** `std.mem.Allocator` - Memory allocator
- **data** `[]const u8` - Data to encode

### Returns

`Blob` - Allocated blob (caller owns memory)

### Errors

- `error.DataTooLarge` - Data exceeds maximum size

### Example

```zig
const Blob = @import("primitives").Blob;

const data = "L2 rollup batch #42";
const blob = try Blob.fromData(allocator, data);
defer blob.deinit(allocator);

// Extract data
const decoded = try blob.toData(allocator);
defer allocator.free(decoded);

std.debug.print("Decoded: {s}\n", .{decoded});
```

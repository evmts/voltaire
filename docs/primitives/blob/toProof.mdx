## Signature

```typescript
blob.toProof(): Proof
```

Generate KZG proof for blob. Proof demonstrates that blob corresponds to its commitment without revealing blob contents.

### Returns

`Proof` - 48-byte KZG proof

### Throws

- If blob size is invalid (not exactly 131,072 bytes)
- If KZG library not available

### Example

```typescript
import { Blob } from '@tevm/voltaire';

const blob = Blob.fromData(data);
const commitment = blob.toCommitment();
const proof = blob.toProof();

console.log(proof.length); // 48 bytes

// Verify proof
const isValid = blob.verify(commitment, proof);
console.log(isValid); // true
```

## Signature

```zig
pub fn toProof(self: Blob, allocator: std.mem.Allocator) !Proof
```

Generate KZG proof for blob.

### Parameters

- **allocator** `std.mem.Allocator` - Memory allocator

### Returns

`Proof` - Allocated 48-byte proof (caller owns memory)

### Errors

- `error.InvalidBlobSize` - Blob size incorrect
- `error.KzgNotAvailable` - KZG library not loaded

### Example

```zig
const Blob = @import("primitives").Blob;

const blob = try Blob.fromData(allocator, data);
defer blob.deinit(allocator);

const proof = try blob.toProof(allocator);
defer proof.deinit(allocator);

std.debug.print("Proof: {} bytes\n", .{proof.len});
```

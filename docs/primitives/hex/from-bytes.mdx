## `Hex.fromBytes(bytes: Uint8Array): BrandedHex`

Converts raw byte array to hex string with `0x` prefix.

**Parameters:**
- `bytes: Uint8Array` - Raw byte array to convert

**Returns:** `BrandedHex` - Hex string with `0x` prefix

**Example:**

```typescript
import { Hex } from '@tevm/voltaire'

// Convert bytes to hex
const bytes = new Uint8Array([0x12, 0x34, 0xAB, 0xCD])
const hex = Hex.fromBytes(bytes)
// "0x1234abcd"

// Empty bytes
const empty = Hex.fromBytes(new Uint8Array())
// "0x"

// Single byte
const single = Hex.fromBytes(new Uint8Array([0xFF]))
// "0xff"
```

Convert byte slice to hex string representation.

## Output Format

All output uses lowercase hex digits:

```typescript
const bytes = new Uint8Array([0xAB, 0xCD, 0xEF])
const hex = Hex.fromBytes(bytes)
// "0xabcdef" (lowercase)
```

## Performance

Efficient byte-to-hex conversion using lookup tables. Suitable for:
- Hash digests (32 bytes)
- Addresses (20 bytes)
- Transaction data
- Cryptographic outputs

**Complexity:** O(n) where n is byte array length

## Common Use Cases

**Convert Hash to Hex:**
```typescript
import { Keccak256 } from '@tevm/voltaire/Keccak256'

const data = new Uint8Array([1, 2, 3])
const hash = Keccak256.hash(data)
const hex = Hex.fromBytes(hash)
```

**Display Binary Data:**
```typescript
const signature = new Uint8Array(65)  // 65-byte signature
crypto.getRandomValues(signature)
const hexSig = Hex.fromBytes(signature)
console.log(hexSig)  // "0x..."
```

**Serialize for RPC:**
```typescript
const rlpEncoded = RLP.encode(transaction)
const hexData = Hex.fromBytes(rlpEncoded)
// Send hexData to JSON-RPC
```

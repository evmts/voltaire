## Behavior

Converts each hex string to bytes, concatenates the byte arrays, then converts back to hex:

```typescript
// Step by step
const hex1 = "0x12"    // [0x12]
const hex2 = "0x3456"  // [0x34, 0x56]
const hex3 = "0x78"    // [0x78]

// Concatenate bytes: [0x12, 0x34, 0x56, 0x78]
const result = Hex.concat(hex1, hex2, hex3)
// "0x12345678"
```

## Common Use Cases

**Build Contract Calldata:**
```typescript
// Function: transfer(address recipient, uint256 amount)
const selector = "0xa9059cbb"  // keccak256("transfer(address,uint256)").slice(0, 4)

// Pad address to 32 bytes (ABI encoding)
const recipient = "0x742d35Cc6634C0532925a3b844Bc9e7595f12345"
const paddedRecipient = Hex.pad(recipient, 32)

// Amount as uint256
const amount = Hex.fromBigInt(1000000000000000000n, 32)  // 1 ETH in wei

// Build calldata
const calldata = Hex.concat(selector, paddedRecipient, amount)
```

**Combine Transaction Components:**
```typescript
const chainId = Hex(1, 32)
const nonce = Hex(42, 32)
const gasPrice = Hex.fromBigInt(20000000000n, 32)
const gasLimit = Hex(21000, 32)
const to = Hex.pad("0x742d35Cc6634C0532925a3b844Bc9e7595f12345", 32)
const value = Hex.fromBigInt(1000000000000000000n, 32)
const data = "0x"

// Concatenate for signing
const rawTx = Hex.concat(chainId, nonce, gasPrice, gasLimit, to, value, data)
```

**Build Storage Keys:**
```typescript
// Mapping storage key: keccak256(key . slot)
const mappingKey = Hex.pad("0x742d35Cc6634C0532925a3b844Bc9e7595f12345", 32)
const slot = Hex(5, 32)
const storageKey = keccak256(Hex.concat(mappingKey, slot))
```

**Combine Signatures:**
```typescript
const r = signature.r  // 32 bytes
const s = signature.s  // 32 bytes
const v = Hex(signature.v, 1)  // 1 byte

const fullSignature = Hex.concat(r, s, v)  // 65 bytes total
```

**Merkle Proof Concatenation:**
```typescript
function verifyProof(leaf: string, proof: string[]): string {
  let current = leaf
  for (const sibling of proof) {
    // Combine and hash
    const combined = Hex.concat(current, sibling)
    current = keccak256(combined)
  }
  return current
}
```

## Empty Strings

Empty hex strings contribute nothing:

```typescript
Hex.concat("0x12", "0x", "0x34")
// "0x1234"

Hex.concat("0x")
// "0x"

Hex.concat("0x", "0x", "0x")
// "0x"
```

## Performance

Allocates single output array of combined size. Complexity: O(n) where n is total byte count across all inputs.

For performance-critical code with many concatenations, consider pre-allocating buffers.

## `Hex.slice(hex: BrandedHex, start: number, end?: number): BrandedHex`

Extracts a portion of hex string by byte indices. Works like `Array.prototype.slice()` but operates on bytes, not hex characters.

**Parameters:**
- `hex: BrandedHex` - Hex string to slice
- `start: number` - Start byte index (inclusive)
- `end?: number` - End byte index (exclusive, optional)

**Returns:** `BrandedHex` - Sliced hex string

**Example:**

```typescript
import { Hex } from '@tevm/voltaire'

const hex = Hex("0x123456789abc")

// Slice from start index
Hex.slice(hex, 0, 2)   // "0x1234"
Hex.slice(hex, 2, 4)   // "0x5678"
Hex.slice(hex, 4)      // "0x9abc" (to end)

// Negative indices (from end)
Hex.slice(hex, -2)     // "0x9abc" (last 2 bytes)
Hex.slice(hex, 0, -2)  // "0x12345678" (all but last 2)

// Extract specific parts
const calldata = Hex("0xa9059cbb" + "00".repeat(64))
const selector = Hex.slice(calldata, 0, 4)  // "0xa9059cbb"
const params = Hex.slice(calldata, 4)       // Parameters
```

## Byte Indices

Operates on **bytes**, not hex characters. Each byte = 2 hex characters:

```typescript
const hex = "0x123456"  // 3 bytes

// Byte index 0 = "0x12"
// Byte index 1 = "0x34"
// Byte index 2 = "0x56"

Hex.slice(hex, 0, 1)  // "0x12" (byte 0)
Hex.slice(hex, 1, 2)  // "0x34" (byte 1)
Hex.slice(hex, 2, 3)  // "0x56" (byte 2)
```

## Common Use Cases

**Extract Function Selector:**
```typescript
const calldata = "0xa9059cbb000000000000000000000000..."
const selector = Hex.slice(calldata, 0, 4)
// "0xa9059cbb"
```

**Extract Parameters:**
```typescript
const calldata = "0xa9059cbb" + params
const allParams = Hex.slice(calldata, 4)  // Skip 4-byte selector
```

**Extract Address from Padded:**
```typescript
// ABI-encoded address (32 bytes, last 20 are address)
const padded = "0x000000000000000000000000742d35Cc6634C0532925a3b844Bc9e7595f12345"
const address = Hex.slice(padded, 12)  // Skip first 12 zero bytes
// "0x742d35Cc6634C0532925a3b844Bc9e7595f12345"
```

**Extract Signature Components:**
```typescript
const signature = "0x" + r + s + v  // 65 bytes
const r = Hex.slice(signature, 0, 32)   // First 32 bytes
const s = Hex.slice(signature, 32, 64)  // Next 32 bytes
const v = Hex.slice(signature, 64, 65)  // Last 1 byte
```

**Parse Storage:**
```typescript
const storage = "0x..." // 32 bytes
const high = Hex.slice(storage, 0, 16)  // High 128 bits
const low = Hex.slice(storage, 16, 32)  // Low 128 bits
```

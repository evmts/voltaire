## `bytecode.validate(): boolean`

Validate bytecode structure. Checks for incomplete PUSH instructions.

**Returns:** `boolean` - `true` if bytecode is structurally valid

**Example:**

```typescript
import { Bytecode } from '@tevm/voltaire'

const valid = Bytecode.fromHex("0x6001")
valid.validate()  // true

const invalid = Bytecode.fromHex("0x60")  // PUSH1 with no data
invalid.validate()  // false
```

## `Bytecode.fromUint8Array(bytes: Uint8Array): Bytecode`

Create bytecode from Uint8Array. Zero-cost conversion with no validation.

**Parameters:**
- `bytes: Uint8Array` - Raw bytecode bytes

**Returns:** `Bytecode` - Bytecode instance with prototype methods

**Example:**

```typescript
import { Bytecode } from '@tevm/voltaire'

// Build bytecode programmatically
const bytes = new Uint8Array([
  0x60, 0x80,  // PUSH1 0x80
  0x60, 0x40,  // PUSH1 0x40
  0x52,        // MSTORE
])
const code = Bytecode.fromUint8Array(bytes)

// Validate structure if needed
if (!code.validate()) {
  throw new Error("Invalid bytecode")
}
```

## `Bytecode.fromUint8Array(allocator: Allocator, bytes: []const u8): !Bytecode`

Create bytecode from raw byte slice. Alias for `init` that emphasizes the byte array input.

**Parameters:**
- `allocator: Allocator` - Memory allocator for analyzed data
- `bytes: []const u8` - Raw bytecode bytes

**Returns:** `Bytecode` - Bytecode with pre-analyzed jump destinations

**Example:**

```zig
const std = @import("std");
const Bytecode = @import("primitives").Bytecode;

var gpa = std.heap.GeneralPurposeAllocator(.{}){};
defer _ = gpa.deinit();
const allocator = gpa.allocator();

const bytes = [_]u8{
  0x60, 0x80,  // PUSH1 0x80
  0x60, 0x40,  // PUSH1 0x40
  0x52,        // MSTORE
};
var bytecode = try Bytecode.fromUint8Array(allocator, &bytes);
defer bytecode.deinit();
```

**Memory:** Must call `deinit()` to clean up.

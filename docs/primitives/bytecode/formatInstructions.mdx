## `bytecode.formatInstructions(): string[]`

Disassemble bytecode into human-readable instruction strings.

**Returns:** `string[]` - Array of formatted instruction strings

**Example:**

```typescript
import { Bytecode } from '@tevm/voltaire'

const code = Bytecode.fromHex("0x60016002015b00")
const disassembly = code.formatInstructions()

console.log(disassembly.join('\n'))
// 0x0000: PUSH1 0x01
// 0x0002: PUSH1 0x02
// 0x0004: ADD
// 0x0005: JUMPDEST
// 0x0006: STOP
```

## `formatInstructions(bytecode: BrandedBytecode): string[]`

Disassemble bytecode into human-readable instruction strings.

**Parameters:**
- `bytecode: BrandedBytecode` - Bytecode to disassemble

**Returns:** `string[]` - Array of formatted instruction strings

**Example:**

```typescript
import * as Bytecode from '@tevm/voltaire/Bytecode'

const code = Bytecode.fromHex("0x60016002015b00")
const disassembly = Bytecode.formatInstructions(code)

console.log(disassembly.join('\n'))
// 0x0000: PUSH1 0x01
// 0x0002: PUSH1 0x02
// 0x0004: ADD
// 0x0005: JUMPDEST
// 0x0006: STOP
```

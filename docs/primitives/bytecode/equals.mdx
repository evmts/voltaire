## `bytecode.equals(other: BrandedBytecode): boolean`

Compare two bytecode instances for equality.

**Parameters:**
- `other: BrandedBytecode` - Bytecode to compare with

**Returns:** `boolean` - `true` if bytecodes are equal

**Example:**

```typescript
import { Bytecode } from '@tevm/voltaire'

const code1 = Bytecode.fromHex("0x6001")
const code2 = Bytecode.fromHex("0x6001")
const code3 = Bytecode.fromHex("0x6002")

console.log(code1.equals(code2))  // true
console.log(code1.equals(code3))  // false
```

## `equals(a: BrandedBytecode, b: BrandedBytecode): boolean`

Compare two bytecode instances for equality.

**Parameters:**
- `a: BrandedBytecode` - First bytecode
- `b: BrandedBytecode` - Second bytecode

**Returns:** `boolean` - `true` if bytecodes are equal

**Example:**

```typescript
import * as Bytecode from '@tevm/voltaire/Bytecode'

const code1 = Bytecode.fromHex("0x6001")
const code2 = Bytecode.fromHex("0x6001")
const code3 = Bytecode.fromHex("0x6002")

console.log(equals(code1, code2))  // true
console.log(equals(code1, code3))  // false
```

## `std.mem.eql(u8, bc1.code, bc2.code) bool`

Compare bytecode by directly comparing the underlying byte arrays.

```zig
const std = @import("std");
const bytecode = @import("bytecode");

var gpa = std.heap.GeneralPurposeAllocator(.{}){};
defer _ = gpa.deinit();
const allocator = gpa.allocator();

const bc1 = try bytecode.Bytecode.init(allocator, &[_]u8{ 0x60, 0x01 });
defer bc1.deinit(allocator);

const bc2 = try bytecode.Bytecode.init(allocator, &[_]u8{ 0x60, 0x01 });
defer bc2.deinit(allocator);

const equal = std.mem.eql(u8, bc1.code, bc2.code);
std.debug.print("{}\n", .{equal});  // true
```

## `bool bytecode_equals(bytecode_t a, bytecode_t b)`

Compare two bytecode instances for equality.

```c
#include "voltaire.h"

uint8_t code1[] = {0x60, 0x01};
uint8_t code2[] = {0x60, 0x01};
uint8_t code3[] = {0x60, 0x02};

bytecode_t bc1 = bytecode_init(code1, 2);
bytecode_t bc2 = bytecode_init(code2, 2);
bytecode_t bc3 = bytecode_init(code3, 2);

bool equal1 = bytecode_equals(bc1, bc2);
bool equal2 = bytecode_equals(bc1, bc3);

printf("%d\n", equal1);  // 1 (true)
printf("%d\n", equal2);  // 0 (false)

bytecode_free(bc1);
bytecode_free(bc2);
bytecode_free(bc3);
```

## `bytecode.stripMetadata(): Bytecode`

Remove Solidity metadata from bytecode.

**Returns:** `Bytecode` - Bytecode with metadata removed

**Example:**

```typescript
import { Bytecode } from '@tevm/voltaire'

const deployed = Bytecode.fromHex(deployedBytecode)
const stripped = deployed.stripMetadata()

// Compare sizes
console.log(`Original: ${deployed.size()} bytes`)
console.log(`Stripped: ${stripped.size()} bytes`)
```

**Defined in:** [primitives/Bytecode/BrandedBytecode/stripMetadata.js](https://github.com/evmts/voltaire/blob/main/src/primitives/Bytecode/BrandedBytecode/stripMetadata.js)

## `stripMetadata(bytecode: BrandedBytecode): BrandedBytecode`

Remove Solidity metadata from bytecode.

**Parameters:**
- `bytecode: BrandedBytecode` - Bytecode to strip

**Returns:** `BrandedBytecode` - Bytecode with metadata removed

**Example:**

```typescript
import * as Bytecode from '@tevm/voltaire/Bytecode'

const deployed = Bytecode.fromHex(deployedBytecode)
const stripped = Bytecode.stripMetadata(deployed)

// Compare sizes
console.log(`Original: ${size(deployed)} bytes`)
console.log(`Stripped: ${size(stripped)} bytes`)
```

**Defined in:** [primitives/Bytecode/BrandedBytecode/stripMetadata.js](https://github.com/evmts/voltaire/blob/main/src/primitives/Bytecode/BrandedBytecode/stripMetadata.js)

## `bytecode.isValidJumpDest(offset: number): boolean`

Check if specific position is valid JUMPDEST.

**Parameters:**
- `offset: number` - Position to check

**Returns:** `boolean` - `true` if position is valid JUMPDEST

**Example:**

```typescript
import { Bytecode } from '@tevm/voltaire'

const code = Bytecode.fromHex("0x5b60005b")  // JUMPDEST, PUSH1 0x00, JUMPDEST

code.isValidJumpDest(0)  // true  - JUMPDEST at start
code.isValidJumpDest(1)  // false - PUSH1 opcode
code.isValidJumpDest(2)  // false - PUSH1 data
code.isValidJumpDest(3)  // true  - JUMPDEST at end
```

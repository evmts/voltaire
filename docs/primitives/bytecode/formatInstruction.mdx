## `Bytecode.formatInstruction(instruction: Instruction): string`

Format single instruction into human-readable string.

**Parameters:**
- `instruction: Instruction` - Instruction object with `{ opcode, position, pushData? }`

**Returns:** `string` - Formatted instruction string

**Example:**

```typescript
import { Bytecode } from '@tevm/voltaire'

const code = Bytecode.fromHex("0x6001")
const instructions = code.parseInstructions()

const formatted = Bytecode.formatInstruction(instructions[0])
console.log(formatted)  // "0x0000: PUSH1 0x01"
```

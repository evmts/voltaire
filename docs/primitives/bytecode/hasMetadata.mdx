## `bytecode.hasMetadata(): boolean`

Check if bytecode contains Solidity compiler metadata.

**Returns:** `boolean` - `true` if metadata is present

**Example:**

```typescript
import { Bytecode } from '@tevm/voltaire'

const deployed = Bytecode.fromHex(deployedBytecode)
if (deployed.hasMetadata()) {
  console.log("Contains Solidity metadata")
  const stripped = deployed.stripMetadata()
}
```

## `hasMetadata(bytecode: BrandedBytecode): boolean`

Check if bytecode contains Solidity compiler metadata.

**Parameters:**
- `bytecode: BrandedBytecode` - Bytecode to check

**Returns:** `boolean` - `true` if metadata is present

**Example:**

```typescript
import * as Bytecode from '@tevm/voltaire/Bytecode'

const deployed = Bytecode.fromHex(deployedBytecode)
if (hasMetadata(deployed)) {
  console.log("Contains Solidity metadata")
  const stripped = Bytecode.stripMetadata(deployed)
}
```

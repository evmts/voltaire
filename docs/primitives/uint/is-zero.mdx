## `Uint.isZero(value: BrandedUint): boolean`

Check if value is zero.

**Parameters:**
- `value: BrandedUint` - Value to check

**Returns:** `boolean` - True if zero

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

Uint.isZero(Uint.ZERO)          // true
Uint.isZero(Uint.from(0n))      // true
Uint.isZero(Uint.from(1n))      // false
Uint.isZero(Uint.MAX)           // false
```

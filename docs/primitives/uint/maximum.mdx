## `Uint.maximum(a: BrandedUint, b: BrandedUint): BrandedUint`

Get maximum of two values.

**Parameters:**
- `a: BrandedUint` - First value
- `b: BrandedUint` - Second value

**Returns:** `BrandedUint` - Larger value

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

Uint.maximum(Uint.from(100n), Uint.from(200n))  // 200
Uint.maximum(Uint.from(200n), Uint.from(100n))  // 200
Uint.maximum(Uint.ZERO, Uint.MAX)               // MAX
```

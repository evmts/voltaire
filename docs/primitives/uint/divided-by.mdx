## `Uint.dividedBy(a: BrandedUint, b: BrandedUint): BrandedUint`

Divide Uint256 values. Floor division (rounds down).

**Parameters:**
- `a: BrandedUint` - Dividend
- `b: BrandedUint` - Divisor

**Returns:** `BrandedUint` - Quotient (floor division)

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

const a = Uint.from(100n)
const b = Uint.from(10n)
Uint.dividedBy(a, b)  // 10

// Floor division
const c = Uint.from(7n)
const d = Uint.from(2n)
Uint.dividedBy(c, d)  // 3 (not 3.5)

// Division by zero throws
Uint.dividedBy(a, Uint.ZERO)  // Error
```

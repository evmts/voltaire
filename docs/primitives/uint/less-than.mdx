## `Uint.lessThan(a: BrandedUint, b: BrandedUint): boolean`

Check if first value is less than second.

**Parameters:**
- `a: BrandedUint` - First value
- `b: BrandedUint` - Second value

**Returns:** `boolean` - True if a < b

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

Uint.lessThan(Uint.from(100n), Uint.from(200n))  // true
Uint.lessThan(Uint.from(200n), Uint.from(100n))  // false
Uint.lessThan(Uint.from(100n), Uint.from(100n))  // false
```

## `Uint.popCount(value: BrandedUint): number`

Count number of set (1) bits in value.

**Parameters:**
- `value: BrandedUint` - Value to analyze

**Returns:** `number` - Number of set bits (0-256)

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

Uint.popCount(Uint.ZERO)        // 0
Uint.popCount(Uint.from(1n))    // 1
Uint.popCount(Uint.from(3n))    // 2 (0b11)
Uint.popCount(Uint.from(15n))   // 4 (0b1111)
Uint.popCount(Uint.MAX)         // 256 (all bits set)
```

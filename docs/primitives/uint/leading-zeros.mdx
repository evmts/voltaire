## `Uint.leadingZeros(value: BrandedUint): number`

Count leading zero bits (from most significant bit).

**Parameters:**
- `value: BrandedUint` - Value to analyze

**Returns:** `number` - Number of leading zeros (0-256)

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

Uint.leadingZeros(Uint.ZERO)       // 256
Uint.leadingZeros(Uint.from(1n))   // 255
Uint.leadingZeros(Uint.from(255n)) // 248
Uint.leadingZeros(Uint.MAX)        // 0
```

## `Uint.shiftRight(value: BrandedUint, bits: number): BrandedUint`

Right shift value by specified number of bits (logical shift).

**Parameters:**
- `value: BrandedUint` - Value to shift
- `bits: number` - Number of bits to shift right

**Returns:** `BrandedUint` - Shifted result

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

const a = Uint.from(256n)
Uint.shiftRight(a, 8)  // 1 (256 >> 8)

const b = Uint.from(240n)
Uint.shiftRight(b, 4)  // 15 (240 >> 4)

// Shifts out all bits
Uint.shiftRight(Uint.from(255n), 256)  // 0
```

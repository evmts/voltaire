## `Uint.toAbiEncoded(value: BrandedUint): Uint8Array`

Convert Uint256 to ABI-encoded bytes (32 bytes, big-endian).

**Parameters:**
- `value: BrandedUint` - Uint256 value

**Returns:** `Uint8Array` - ABI-encoded 32 bytes

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

const value = Uint.from(255n)
const encoded = Uint.toAbiEncoded(value)
// Uint8Array(32) [0, 0, ..., 0, 255]

// Always 32 bytes
encoded.length  // 32
```

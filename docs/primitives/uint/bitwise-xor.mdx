## `Uint.bitwiseXor(a: BrandedUint, b: BrandedUint): BrandedUint`

Bitwise XOR (exclusive OR) operation.

**Parameters:**
- `a: BrandedUint` - First value
- `b: BrandedUint` - Second value

**Returns:** `BrandedUint` - Bitwise XOR result

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

const a = Uint.from(0b1100n)
const b = Uint.from(0b1010n)
Uint.bitwiseXor(a, b)  // 0b0110 (6)
```

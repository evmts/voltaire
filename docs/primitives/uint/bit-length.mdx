## `Uint.bitLength(value: BrandedUint): number`

Get number of bits required to represent value (position of highest set bit + 1).

**Parameters:**
- `value: BrandedUint` - Value to measure

**Returns:** `number` - Bit length (0-256)

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

Uint.bitLength(Uint.from(0n))     // 0
Uint.bitLength(Uint.from(1n))     // 1
Uint.bitLength(Uint.from(2n))     // 2
Uint.bitLength(Uint.from(255n))   // 8
Uint.bitLength(Uint.from(256n))   // 9
Uint.bitLength(Uint.MAX)          // 256
```

## `Uint.bitwiseOr(a: BrandedUint, b: BrandedUint): BrandedUint`

Bitwise OR operation.

**Parameters:**
- `a: BrandedUint` - First value
- `b: BrandedUint` - Second value

**Returns:** `BrandedUint` - Bitwise OR result

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

const a = Uint.from(0b1100n)
const b = Uint.from(0b1010n)
Uint.bitwiseOr(a, b)  // 0b1110 (14)
```

## `Uint.minimum(a: BrandedUint, b: BrandedUint): BrandedUint`

Get minimum of two values.

**Parameters:**
- `a: BrandedUint` - First value
- `b: BrandedUint` - Second value

**Returns:** `BrandedUint` - Smaller value

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

Uint.minimum(Uint.from(100n), Uint.from(200n))  // 100
Uint.minimum(Uint.from(200n), Uint.from(100n))  // 100
Uint.minimum(Uint.ZERO, Uint.MAX)               // ZERO
```

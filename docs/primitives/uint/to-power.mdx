## `Uint.toPower(base: BrandedUint, exponent: BrandedUint): BrandedUint`

Raise to power. Wraps on overflow (mod 2^256).

**Parameters:**
- `base: BrandedUint` - Base value
- `exponent: BrandedUint` - Exponent

**Returns:** `BrandedUint` - Result (mod 2^256)

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

const base = Uint.from(2n)
const exp = Uint.from(8n)
Uint.toPower(base, exp)  // 256 (2^8)

// Large exponents wrap
Uint.toPower(Uint.from(2n), Uint.from(256n))  // 0 (wraps)
```

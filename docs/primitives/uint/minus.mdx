## `Uint.minus(a: BrandedUint, b: BrandedUint): BrandedUint`

Subtract Uint256 values. Wraps on underflow (mod 2^256).

**Parameters:**
- `a: BrandedUint` - Minuend
- `b: BrandedUint` - Subtrahend

**Returns:** `BrandedUint` - Difference (mod 2^256)

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

const a = Uint.from(100n)
const b = Uint.from(50n)
Uint.minus(a, b)  // 50

// Underflow wraps
Uint.minus(Uint.ZERO, Uint.ONE)  // MAX
Uint.minus(Uint.from(10n), Uint.from(20n))  // MAX - 9
```

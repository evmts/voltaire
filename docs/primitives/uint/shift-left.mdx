## `Uint.shiftLeft(value: BrandedUint, bits: number): BrandedUint`

Left shift value by specified number of bits. Wraps on overflow.

**Parameters:**
- `value: BrandedUint` - Value to shift
- `bits: number` - Number of bits to shift left

**Returns:** `BrandedUint` - Shifted result (mod 2^256)

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

const a = Uint.from(1n)
Uint.shiftLeft(a, 8)   // 256 (1 << 8)

const b = Uint.from(15n)
Uint.shiftLeft(b, 4)   // 240 (15 << 4)

// Wraps on overflow
Uint.shiftLeft(Uint.from(1n), 256)  // 0 (wraps)
```

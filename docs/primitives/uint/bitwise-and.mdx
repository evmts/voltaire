## `Uint.bitwiseAnd(a: BrandedUint, b: BrandedUint): BrandedUint`

Bitwise AND operation.

**Parameters:**
- `a: BrandedUint` - First value
- `b: BrandedUint` - Second value

**Returns:** `BrandedUint` - Bitwise AND result

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

const a = Uint.from(0b1100n)
const b = Uint.from(0b1010n)
Uint.bitwiseAnd(a, b)  // 0b1000 (8)

// Masking
const value = Uint.from(0xffn)
const mask = Uint.from(0x0fn)
Uint.bitwiseAnd(value, mask)  // 0x0f
```

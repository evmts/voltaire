## `Uint.isValid(value: unknown): value is BrandedUint`

Type guard to check if value is valid BrandedUint.

**Parameters:**
- `value: unknown` - Value to check

**Returns:** `value is BrandedUint` - True if valid

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

const a = Uint.from(100n)
Uint.isValid(a)  // true

Uint.isValid(100n)           // false
Uint.isValid("100")          // false
Uint.isValid(new Uint8Array(32))  // false (not branded)
Uint.isValid(null)           // false
```

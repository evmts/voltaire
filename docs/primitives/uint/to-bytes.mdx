## `Uint.toBytes(value: BrandedUint): Uint8Array`

Convert Uint256 to 32-byte array (big-endian).

**Parameters:**
- `value: BrandedUint` - Uint256 value

**Returns:** `Uint8Array` - 32-byte array (big-endian)

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

const a = Uint.from(255n)
Uint.toBytes(a)  // Uint8Array(32) [0, 0, ..., 0, 255]

const b = Uint.from(256n)
Uint.toBytes(b)  // Uint8Array(32) [0, 0, ..., 1, 0]

// Always 32 bytes
const c = Uint.from(0n)
Uint.toBytes(c).length  // 32
```

<Tab title="Zig">

## `uint.toBytes(): [32]u8`

Convert to 32-byte array (big-endian).

**Example:**

```zig
const Uint = @import("primitives").Uint;

const value = Uint.init(255);
const bytes = value.toBytes();
// bytes[31] == 255
```

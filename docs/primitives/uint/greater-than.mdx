## `Uint.greaterThan(a: BrandedUint, b: BrandedUint): boolean`

Check if first value is greater than second.

**Parameters:**
- `a: BrandedUint` - First value
- `b: BrandedUint` - Second value

**Returns:** `boolean` - True if a > b

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

Uint.greaterThan(Uint.from(200n), Uint.from(100n))  // true
Uint.greaterThan(Uint.from(100n), Uint.from(200n))  // false
Uint.greaterThan(Uint.from(100n), Uint.from(100n))  // false
```

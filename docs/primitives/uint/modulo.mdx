## `Uint.modulo(a: BrandedUint, b: BrandedUint): BrandedUint`

Get remainder of division.

**Parameters:**
- `a: BrandedUint` - Dividend
- `b: BrandedUint` - Modulus

**Returns:** `BrandedUint` - Remainder

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

const a = Uint.from(17n)
const b = Uint.from(5n)
Uint.modulo(a, b)  // 2 (17 = 3*5 + 2)

// Even/odd check
Uint.modulo(Uint.from(100n), Uint.from(2n))  // 0
Uint.modulo(Uint.from(101n), Uint.from(2n))  // 1

// Modulo by zero throws
Uint.modulo(a, Uint.ZERO)  // Error
```

## `Uint.equals(a: BrandedUint, b: BrandedUint): boolean`

Check if two Uint256 values are equal.

**Parameters:**
- `a: BrandedUint` - First value
- `b: BrandedUint` - Second value

**Returns:** `boolean` - True if equal

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

const a = Uint.from(100n)
const b = Uint.from(100n)
Uint.equals(a, b)  // true

const c = Uint.from(200n)
Uint.equals(a, c)  // false

Uint.equals(Uint.ZERO, Uint.from(0n))  // true
```

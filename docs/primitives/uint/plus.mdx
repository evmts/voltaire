## `Uint.plus(a: BrandedUint, b: BrandedUint): BrandedUint`

Add two Uint256 values. Wraps on overflow (mod 2^256).

**Parameters:**
- `a: BrandedUint` - First value
- `b: BrandedUint` - Second value

**Returns:** `BrandedUint` - Sum (mod 2^256)

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

const a = Uint.from(100n)
const b = Uint.from(50n)
Uint.plus(a, b)  // 150

// Overflow wraps
Uint.plus(Uint.MAX, Uint.ONE)  // 0 (wraps)
Uint.plus(Uint.MAX, Uint.from(2n))  // 1
```

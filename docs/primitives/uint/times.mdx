## `Uint.times(a: BrandedUint, b: BrandedUint): BrandedUint`

Multiply Uint256 values. Wraps on overflow (mod 2^256).

**Parameters:**
- `a: BrandedUint` - First value
- `b: BrandedUint` - Second value

**Returns:** `BrandedUint` - Product (mod 2^256)

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

const a = Uint.from(10n)
const b = Uint.from(5n)
Uint.times(a, b)  // 50

// Overflow wraps
const large = Uint.from(2n ** 200n)
Uint.times(large, large)  // Wraps (result mod 2^256)
```

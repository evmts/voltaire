## `Uint.toString(value: BrandedUint, radix?: number): string`

Convert Uint256 to string. Supports radix 2-36 (default 10).

**Parameters:**
- `value: BrandedUint` - Uint256 value
- `radix?: number` - Base (2-36, default 10)

**Returns:** `string` - String representation

**Example:**

```typescript
import { Uint } from '@tevm/voltaire'

const value = Uint.from(255n)

Uint.toString(value)      // "255" (decimal)
Uint.toString(value, 10)  // "255" (decimal)
Uint.toString(value, 16)  // "ff" (hex, no prefix)
Uint.toString(value, 2)   // "11111111" (binary)
Uint.toString(value, 8)   // "377" (octal)
```

## Radix Support

### Decimal (base 10)

```typescript
Uint.toString(Uint.from(12345n))  // "12345"
```

### Hexadecimal (base 16)

```typescript
Uint.toString(Uint.from(255n), 16)  // "ff" (no 0x prefix)
```

<Tip>
For hex with `0x` prefix, use `toHex()` instead.
</Tip>

### Binary (base 2)

```typescript
Uint.toString(Uint.from(15n), 2)  // "1111"
```

### Octal (base 8)

```typescript
Uint.toString(Uint.from(64n), 8)  // "100"
```

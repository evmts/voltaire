## Output Format

Hex strings are **minimal** - no unnecessary leading zeros:

```typescript
Uint.toHex(Uint.from(0n))      // "0x0"
Uint.toHex(Uint.from(1n))      // "0x1"
Uint.toHex(Uint.from(15n))     // "0xf"
Uint.toHex(Uint.from(16n))     // "0x10"
Uint.toHex(Uint.from(255n))    // "0xff"
Uint.toHex(Uint.from(256n))    // "0x100"
```

### Always Lowercase

```typescript
Uint.toHex(Uint.from(0xdeadbeefn))  // "0xdeadbeef"
```

### Always Has 0x Prefix

```typescript
// Never returns without prefix
Uint.toHex(Uint.from(255n))  // "0xff" (not "ff")
```

## Usage Patterns

### Logging and Debugging

```typescript
const value = Uint.from(12345n)
console.log(`Value: ${Uint.toHex(value)}`)  // "Value: 0x3039"
```

### Ethereum RPC

```typescript
// Format for JSON-RPC
const balance = Uint.from(1000000000000000000n)
const rpcValue = Uint.toHex(balance)  // "0xde0b6b3a7640000"

await eth_sendTransaction({
  value: rpcValue
})
```

### Storage Keys

```typescript
const slot = Uint.from(5n)
const key = Uint.toHex(slot)  // "0x5"
```

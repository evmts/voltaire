# BrandedUint256 Type

`BrandedUint256` is the underlying branded type for Uint256 operations used throughout Voltaire's Uint module.

## Type Definition

```typescript
export type BrandedUint256 = bigint & {
  readonly __tag: "Uint256"
}
```

Runtime-validated `bigint` constrained to `0 <= value <= 2^256-1`. The brand prevents accidental mixing with regular bigints.

## Usage

Use through the `Uint` class:

```typescript
import { Uint } from '@tevm/voltaire'

// Create Uint
const a = Uint.from(100n)
const b = Uint.fromHex("0xff")

// Arithmetic
const sum = Uint.plus(a, b)
const product = Uint.times(a, b)

// Conversions
const hex = Uint.toHex(sum)
const bytes = Uint.toBytes(sum)
```

## Constants

```typescript
Uint.MAX   // 2^256 - 1 (largest Uint256)
Uint.MIN   // 0 (smallest Uint256)
Uint.ZERO  // 0
Uint.ONE   // 1
Uint.SIZE  // 32 (bytes)
```

## Opcode.isDup()

Check if opcode is DUP1-DUP16.

<Tabs>
<Tab title="Namespace API">
```typescript
import * as Opcode from '@tevm/voltaire/Opcode'

Opcode.isDup(0x80)  // true (DUP1)
Opcode.isDup(0x8F)  // true (DUP16)
Opcode.isDup(0x60)  // false (PUSH1)
Opcode.isDup(0x90)  // false (SWAP1)
```
</Tab>
<Tab title="Functional API">
```typescript
import { isDup, DUP1, DUP16 } from '@tevm/voltaire/Opcode'

isDup(DUP1)   // true
isDup(DUP16)  // true
```
</Tab>
<Tab title="Zig">
```zig
const Opcode = @import("primitives").opcode;

const is_dup1 = Opcode.isDup(0x80);   // true
const is_push1 = Opcode.isDup(0x60);  // false
```
</Tab>
</Tabs>

## Parameters

- `opcode: BrandedOpcode` - Opcode to check

## Returns

`boolean` - True if DUP1-DUP16 (0x80-0x8F)

## DUP Operations

Duplicate stack item at depth 1-16:
- **DUP1** (0x80): Duplicate top item
- **DUP2** (0x81): Duplicate 2nd item
- **DUP16** (0x8F): Duplicate 16th item

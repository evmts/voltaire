BLAKE2F precompile (0x09)

- rounds (4 bytes, big-endian) - number of compression rounds
- h (64 bytes) - state vector
- m (128 bytes) - message block
- t (16 bytes) - offset counters
- f (1 byte) - final block flag (must be 0 or 1)

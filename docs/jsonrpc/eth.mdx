# eth Methods

The `eth` namespace provides standard Ethereum JSON-RPC methods for interacting with Ethereum nodes.

## Overview

**Total Methods**: 40

The `eth` namespace covers:
- **Block queries** - Get block data by hash or number
- **Transaction operations** - Send, get, and query transactions
- **State queries** - Get balances, code, storage, and account state
- **Filters & logs** - Create filters and query event logs
- **Gas estimation** - Estimate gas costs for transactions
- **Network info** - Chain ID, sync status, coinbase

## Common Methods

### eth_blockNumber
Get the current block number.

### eth_call
Execute a message call (read-only contract call).

### eth_getBalance
Get the balance of an address at a given block.

### eth_getBlockByNumber / eth_getBlockByHash
Get block data by block number or hash.

### eth_getTransactionByHash
Get transaction data by transaction hash.

### eth_getTransactionReceipt
Get transaction receipt (includes logs and status).

### eth_getLogs
Get logs matching a filter.

### eth_sendRawTransaction
Submit a signed transaction to the network.

### eth_estimateGas
Estimate gas required for a transaction.

## Usage

```typescript
import * as eth from '@tevm/voltaire/jsonrpc/eth';

// eth_getBalance
const balanceParams: typeof eth.eth_getBalance.Params = {
  address: '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb0',
  block: 'latest'
};

// eth_call
const callParams: typeof eth.eth_call.Params = {
  to: '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb0',
  data: '0x70a08231...',
  block: 'latest'
};
```

# Address

20-byte Ethereum address type extending Uint8Array

## Overview

[Branded](/primitives/branded-types) `Uint8Array` (20 bytes) representing Ethereum address. Zero-overhead design supports both tree-shakeable namespace methods and class instances.

## Quick Start

<Tabs>
<TabItem label="Class API">
```typescript
import { Address } from '@tevm/voltaire'

// Construct from various formats (see Constructors)
const addr = new Address("0x742d35Cc6634C0532925a3b844Bc9e7595f51e3e")
const addr2 = Address.fromBytes(new Uint8Array(20))

// Validate (see Validation)
if (!Address.isValid("0xinvalid")) throw new Error("Invalid address")

// Convert formats (see Conversions)
addr.toHex()          // "0x742d35cc6634c0532925a3b844bc9e7595f51e3e"
addr.toChecksummed()  // "0x742d35Cc6634C0532925a3b844Bc9e7595f51e3e"
addr.toShortHex()     // "0x742d…1e3e"

// Compare (see Comparisons)
addr.equals(addr2)    // false
addr.isZero()         // false

// Calculate contract addresses (see Contract Addresses)
const contractAddr = addr.calculateCreateAddress(0n)
const create2Addr = addr.calculateCreate2Address(salt, initCode)
```
</TabItem>
<TabItem label="Namespace API (Tree-shakeable)">
```typescript
import { Address } from '@tevm/voltaire'

// Tree-shakeable imports minimize bundle size
const addr = Address.fromHex("0x742d35Cc6634C0532925a3b844Bc9e7595f51e3e")

// Each function imported independently
Address.toChecksummed(addr)  // "0x742d35Cc6634C0532925a3b844Bc9e7595f51e3e"
Address.equals(addr, addr2)  // boolean
Address.isZero(addr)         // boolean

// See BrandedAddress for complete functional API
```
</TabItem>

<Aside type="tip" title="Why Uint8Array over hex strings?">
Uint8Array eliminates casing bugs (checksummed vs lowercase). Conversions happen at boundaries; internal representation stays byte-based. See [Bytes](/primitives/bytes) for byte utilities.
</Aside>

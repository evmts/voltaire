Class RateLimiter

Token Bucket Rate Limiter

Implements token bucket algorithm for rate limiting. Requests consume tokens from a bucket that refills over time. When bucket is empty, requests are queued, rejected, or dropped based on strategy.

Example

// Limit to 10 requests per second
const limiter = new RateLimiter({
  maxRequests: 10,
  interval: 1000,
  strategy: 'queue'
});

// Execute with rate limit
const result = await limiter.execute(() => provider.eth_blockNumber());

// Wrap function with rate limiter
const getBalance = limiter.wrap(
  (address: string) => provider.eth_getBalance(address)
);
const balance = await getBalance('0x123...');

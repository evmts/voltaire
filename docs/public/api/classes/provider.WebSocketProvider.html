Class WebSocketProvider

WebSocket Provider implementation

Implements Provider interface using WebSocket transport for real-time communication. Supports native pub/sub subscriptions for events.

Example

const provider = new WebSocketProvider({
  url: 'wss://eth.example.com',
  reconnect: true
});

await provider.connect();

const blockNumber = await provider.eth_blockNumber();
console.log('Block:', blockNumber.result);

Ed25519 Digital Signature Algorithm

Edwards-curve Digital Signature Algorithm (EdDSA) using Curve25519. Fast, secure, and deterministic signatures without requiring a hash function. Used in many modern protocols including SSH, TLS 1.3, and cryptocurrency.

// Sign a message

const valid = Ed25519.verify(signature, message, keypair.publicKey);

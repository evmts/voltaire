Gas cost constants for EVM operations Based on Yellow Paper Appendix G and various EIPs

Type declaration

  • ReadonlyBALANCE: 2600n

    Cold BALANCE access (EIP-2929)

  • ReadonlyBASE: 2n

    Base cost for most opcodes

  • ReadonlyBLOCKHASH: 20n

    BLOCKHASH opcode

  • ReadonlyCALL: 100n

    Base gas for message call

  • ReadonlyCALL_STIPEND: 2300n

    Gas stipend provided to called contract when value > 0

  • ReadonlyCALL_VALUE: 9000n

    Additional cost for non-zero value transfer

  • ReadonlyCALLDATA_NONZERO: 16n

    Non-zero byte in calldata

  • ReadonlyCALLDATA_ZERO: 4n

    Zero byte in calldata (cheaper)

  • ReadonlyCOLD_ACCOUNT_ACCESS: 2600n

    Cold account access cost (EIP-2929)

  • ReadonlyCOPY: 3n

    Memory/storage copy cost per word

  • ReadonlyCREATE: 32000n

    Contract creation base cost

  • ReadonlyEXP: 10n

    EXP base cost

  • ReadonlyEXP_BYTE: 50n

    EXP per byte cost

  • ReadonlyEXTCODECOPY: 2600n

    Cold EXTCODECOPY access (EIP-2929)

  • ReadonlyHIGH: 10n

    High cost operations

  • ReadonlyJUMPDEST: 1n

    JUMPDEST cost

  • ReadonlyLOG: 375n

    LOG0 base cost

  • ReadonlyLOG_DATA: 8n

    Cost per byte of LOG data

  • ReadonlyLOG_TOPIC: 375n

    Cost per LOG topic

  • ReadonlyLOW: 5n

    Low cost operations (MUL, DIV, etc)

  • ReadonlyMEMORY: 3n

    Memory expansion cost per word

  • ReadonlyMID: 8n

    Mid cost operations

  • ReadonlySELFDESTRUCT: 5000n

    SELFDESTRUCT cost (no refund post-London)

  • ReadonlySHA3: 30n

    SHA3/KECCAK256 base cost

  • ReadonlySHA3_WORD: 6n

    SHA3/KECCAK256 per word cost

  • ReadonlySLOAD: 2100n

    Cold SLOAD cost (EIP-2929)

  • ReadonlySLOAD_WARM: 100n

    Cost of SLOAD from warm storage

  • ReadonlySSTORE_CLEAR: 15000n

    SSTORE clear refund (pre-London: 15000, post-London: none)

  • ReadonlySSTORE_RESET: 5000n

    SSTORE from non-zero to non-zero

  • ReadonlySSTORE_SET: 20000n

    SSTORE from zero to non-zero (most expensive)

  • ReadonlyTRANSACTION: 21000n

    Base transaction cost (21000 gas)

  • ReadonlyVERY_LOW: 3n

    Very low cost operations (ADD, SUB, etc)

  • ReadonlyWARM_STORAGE_READ: 100n

    Warm storage access cost (EIP-2929)

---
title: Uint.toBytes
description: Convert Uint256 to byte array in big-endian format
---


<Tabs>
<Tab title="Effect Schema">

```ts
import { UintSchema } from '@tevm/voltaire/Uint/effect'

const u = UintSchema.from(1n)
u.toBytes() // Uint8Array(32) [..., 1]
```

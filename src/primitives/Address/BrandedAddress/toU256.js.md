# toU256

Converts Address to uint256 bigint.

## Signature

```typescript
function toU256(address: BrandedAddress): bigint
```

## Parameters

- `address` - Address bytes to convert

## Returns

Bigint representation of address

## Example

```typescript
const value = Address.toU256(addr);
// 663866926832282913817648144953575950485970711011n
```

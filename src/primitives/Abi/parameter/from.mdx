## Overview

`Parameter.from` is a lightweight helper that brands a parameter object for TypeScript type inference. It does **not** validate types at runtime.

## Quick Start

```typescript
import * as Parameter from '@tevm/voltaire/Abi/parameter';

const amount = Parameter.from({ type: 'uint256', name: 'amount' });
const owner = Parameter.from({ type: 'address', name: 'owner' });
```

## Tuples and Components

```typescript
import * as Parameter from '@tevm/voltaire/Abi/parameter';

const position = Parameter.from({
  type: 'tuple',
  name: 'position',
  components: [
    { type: 'uint256', name: 'amount' },
    { type: 'uint256', name: 'shares' }
  ]
});
```

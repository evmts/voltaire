# BLS12-381 G1 MSM

BLS12-381 G1 curve multi-scalar multiplication (MSM) for efficient batch operations.

## Overview

The BLS12-381 G1 MSM precompile provides:
- **Multi-Scalar Multiplication**: Compute Σ(kᵢ * Pᵢ) efficiently
- **Batch Operations**: More efficient than individual multiplications
- **Variable Gas**: Cost scales with number of points
- **C Library**: Uses blst for performance
- **Address**: 0x000000000000000000000000000000000000000D
- **Since**: Prague (EIP-2537)

**Source**: `src/precompiles/bls12_g1_msm.zig`

## Key Functions

### execute

```zig
const precompiles = @import("precompiles");

pub fn execute(
    allocator: std.mem.Allocator,
    input: []const u8,
    gas_limit: u64,
) PrecompileError!PrecompileResult
```

**Parameters:**
- `allocator`: Memory allocator for result
- `input`: Multiple of 160 bytes (k pairs of point + scalar)
- `gas_limit`: Maximum gas available

**Input Format** (160k bytes for k pairs):
Each pair (160 bytes):
- Bytes 0-127: G1 point
- Bytes 128-159: Scalar (32 bytes)

**Returns:** 128 bytes (result point)

**Errors:** `OutOfGas`, `InvalidInput`, `InvalidPoint`

## Gas Costs

**Formula:** `12000 * k * discount(k)`

Where discount depends on batch size for efficiency.

## Performance

**Native (ReleaseFast)**: Scales sub-linearly with k

# Hash

32-byte cryptographic hash type extending Uint8Array

## Overview

[Branded](/primitives/branded-types) `Uint8Array` (32 bytes) representing cryptographic hash. Zero-overhead design supports both tree-shakeable namespace methods and class instances.

## Quick Start

<Tabs>
<TabItem label="Class API">
```typescript
import { Hash } from '@tevm/voltaire'

// Construct from various formats (see Constructors)
const hash = new Hash("0x1234567890abcdef1234567890abcdef1234567890abcdef1234567890abcdef")
const hash2 = Hash.fromBytes(new Uint8Array(32))

// Hash data with Keccak-256 (see Hashing)
const dataHash = Hash.keccak256(new Uint8Array([1, 2, 3]))
const stringHash = Hash.keccak256String("hello")
const hexHash = Hash.keccak256Hex("0x1234")

// Validate (see Validation)
if (!Hash.isValidHex("0xinvalid")) throw new Error("Invalid hash")

// Convert formats (see Conversions)
hash.toHex()      // "0x1234567890abcdef..."
hash.toString()   // "0x1234567890abcdef..."
hash.format()     // "0x123456...abcdef"

// Compare (see Comparisons)
hash.equals(hash2)    // false
hash.isZero()         // false

// Utilities (see Utilities)
const randomHash = Hash.random()
const copy = hash.clone()
const partial = hash.slice(0, 4)
```
</TabItem>
<TabItem label="Namespace API (Tree-shakeable)">
```typescript
import { Hash } from '@tevm/voltaire'

// Tree-shakeable imports minimize bundle size
const hash = Hash.fromHex("0x1234567890abcdef1234567890abcdef1234567890abcdef1234567890abcdef")

// Each function imported independently
Hash.toHex(hash)     // "0x1234567890abcdef..."
Hash.equals(hash, hash2)  // boolean
Hash.isZero(hash)    // boolean

// Keccak-256 hashing
const dataHash = Hash.keccak256(data)
const stringHash = Hash.keccak256String("hello")

// See BrandedHash for complete functional API
```
</TabItem>
</Tabs>

## Constants

Hash module exports size constants for validation and buffer allocation:

```typescript
import { Hash } from '@tevm/voltaire'

// Size in bytes
Hash.SIZE // 32

// Zero hash constant
Hash.ZERO // Uint8Array(32) [0, 0, 0, ...]

// Usage examples
const buffer = new Uint8Array(Hash.SIZE)  // 32-byte buffer
const isZero = Hash.equals(hash, Hash.ZERO)  // check zero hash
```

**Available constants:**
- `Hash.SIZE = 32` - Hash size in bytes (256 bits)
- `Hash.ZERO` - Zero hash (32 zero bytes)

Source: [constants.js](https://github.com/evmts/primitives/blob/main/src/primitives/Hash/BrandedHash/constants.js)

<Aside type="tip" title="Why Uint8Array over hex strings?">
Uint8Array provides efficient byte operations and eliminates encoding overhead. Conversions happen at boundaries; internal representation stays byte-based. See [Bytes](/primitives/bytes) for byte utilities.
</Aside>

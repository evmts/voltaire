# Comparisons

Methods for comparing Address instances for equality and ordering.

## Equality

### `instance.equals(other)`

Byte-by-byte equality comparison.

```typescript
const addr1 = new Address(69n)
const addr2 = new Address(69n)
const addr3 = Address.fromHex("0x0000000000000000000000000000000000000045")

addr1.equals(addr2) // true
addr1.equals(addr3) // true (69 = 0x45)

const different = new Address(42n)
addr1.equals(different) // false
```

**Parameters:**
- `other: BrandedAddress` - Address to compare

**Returns:** `boolean`

**Static form:**
```typescript
Address.equals(addr1, addr2) // boolean
```

### `instance.isZero()`

Checks if all bytes are zero.

```typescript
Address.zero().isZero() // true
new Address(0n).isZero() // true
new Address(69n).isZero() // false

const zeroHex = Address.fromHex("0x0000000000000000000000000000000000000000")
zeroHex.isZero() // true
```

**Returns:** `boolean`

**Static form:**
```typescript
Address.isZero(addr) // boolean
```

## Ordering

### `instance.compare(other)`

Lexicographic comparison: -1 (less), 0 (equal), 1 (greater).

```typescript
const addr1 = new Address(10n)
const addr2 = new Address(20n)
const addr3 = new Address(10n)

addr1.compare(addr2) // -1 (addr1 < addr2)
addr2.compare(addr1) // 1  (addr2 > addr1)
addr1.compare(addr3) // 0  (addr1 == addr3)

// Use for sorting
const addresses = [addr2, addr1, addr3]
addresses.sort((a, b) => a.compare(b))
```

**Parameters:**
- `other: BrandedAddress` - Address to compare

**Returns:** `number` (-1, 0, or 1)

**Static form:**
```typescript
Address.compare(addr1, addr2) // number
```

### `instance.lessThan(other)`

Returns true if this address is less than other.

```typescript
const addr1 = new Address(10n)
const addr2 = new Address(20n)

if (addr1.lessThan(addr2)) {
  // addr1 comes before addr2
  console.log("10 < 20") // true
}

addr2.lessThan(addr1) // false
```

**Parameters:**
- `other: BrandedAddress` - Address to compare

**Returns:** `boolean`

**Static form:**
```typescript
Address.lessThan(addr1, addr2) // boolean
```

## Usage Examples

### Sorting Addresses

```typescript
const addresses = [
  Address.fromHex("0x742d35Cc6634C0532925a3b844Bc9e7595f51e3e"),
  Address.fromHex("0x0000000000000000000000000000000000000001"),
  Address.fromHex("0xffffffffffffffffffffffffffffffffffffffff"),
]

// Sort ascending
addresses.sort((a, b) => a.compare(b))

// Sort descending
addresses.sort((a, b) => b.compare(a))
```

### Filtering Addresses

```typescript
const addresses = [
  Address.zero(),
  new Address(42n),
  new Address(0n),
  new Address(69n),
]

// Remove zero addresses
const nonZero = addresses.filter(addr => !addr.isZero())

// Find addresses greater than threshold
const threshold = new Address(50n)
const large = addresses.filter(addr => addr.greaterThan(threshold))
```

### Deduplication

```typescript
const addresses = [
  new Address(69n),
  new Address(42n),
  new Address(69n), // duplicate
]

const unique = addresses.filter((addr, i, arr) =>
  arr.findIndex(a => a.equals(addr)) === i
)
```

```typescript

import type { BrandedAddress } from '@tevm/voltaire/BrandedAddress'

const addr1: BrandedAddress = /* ... */

compare(addr1, addr2)      // -1 | 0 | 1

```

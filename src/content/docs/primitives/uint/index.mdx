# Uint (Uint256)

256-bit unsigned integer type with wrapping arithmetic and comprehensive operations

## Overview

[Branded](/primitives/branded-types) `bigint` type representing unsigned 256-bit integer (0 to 2^256-1). Zero-overhead design supports both tree-shakeable namespace methods and class instances with automatic wrapping on overflow.

<Aside type="tip" title="Automatic Wrapping">
Arithmetic operations automatically wrap on overflow/underflow (mod 2^256), matching EVM behavior. Use validation to check bounds before operations if needed.
</Aside>

## Overflow Behavior

All arithmetic and bitwise operations wrap on overflow:

```typescript
import { Uint } from '@tevm/voltaire'

// Overflow wraps around
const max = Uint.MAX
const overflow = max.plus(Uint.ONE)  // Wraps to 0

// Underflow wraps around
const zero = Uint.ZERO
const underflow = zero.minus(Uint.ONE)  // Wraps to MAX

// Multiplication wraps
const large = Uint.from(2n ** 200n)
const product = large.times(large)  // Result mod 2^256

// Shift operations wrap
const value = Uint.ONE
const shifted = value.shiftLeft(260)  // Wraps, keeps only lower 256 bits
```

This matches Solidity and EVM arithmetic behavior. See [Arithmetic](/primitives/uint/arithmetic) for details.

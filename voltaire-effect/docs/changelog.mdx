<Update label="v0.0.1" description="Initial release">
**Added**
- Effect Schema wrappers for all [Voltaire](https://voltaire.tevm.sh) primitives
- Typed errors with `_tag` discrimination
- Services: Provider, Signer, Transport, Contract, Account
- Crypto services: Keccak256, Secp256k1, BLS12-381, HDWallet, BIP-39, and more
- Full test layer support for mocking
- Comprehensive documentation with examples
</Update>

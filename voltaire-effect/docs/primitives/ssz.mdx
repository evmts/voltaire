title: Ssz
description: SSZ (Simple Serialize) encoded data
---

```typescript
import { Ssz } from 'voltaire-effect/primitives'
import * as Schema from 'effect/Schema'

// Schema wraps Uint8Array with branded type
const sszData = Schema.decodeSync(Ssz.Schema)(new Uint8Array([1, 2, 3]))

// Type guard
Schema.is(Ssz.Schema)(new Uint8Array([1, 2, 3])) // true
Schema.is(Ssz.Schema)('not bytes')               // false

// Compose with other schemas
const BeaconDataSchema = Schema.Struct({
  slot: Slot.Schema,
  body: Ssz.Schema,
})
```

Serialization format for Ethereum consensus layer. Branded type wrapper for type safety.

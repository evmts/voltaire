---
title: Uint
description: 256-bit unsigned integer (Uint256)
---

256-bit unsigned integer. `U256` is an alias with renamed exports.

```typescript
import { Uint } from 'voltaire-effect'
import * as Schema from 'effect/Schema'
import { Effect } from 'effect'

// Schema: accepts bigint, number, string
const value = Schema.decodeSync(Uint.BigInt)(1000000000000000000n)

// Hex and bytes
const fromHex = Schema.decodeSync(Uint.FromHexSchema)('0xde0b6b3a7640000')
const fromBytes = Schema.decodeSync(Uint.FromBytesSchema)(
  new Uint8Array([0x0d, 0xe0, 0xb6, 0xb3, 0xa7, 0x64, 0x00, 0x00])
)

// Effect
const program = Uint.from(1000000000000000000n)
// Effect.Effect<Uint256Type, UintError>
```

## U256 Alias

```typescript
import { U256 } from 'voltaire-effect'

const value = Schema.decodeSync(U256.BigInt)(1n)
```

## Errors

Fails on overflow (> 2²⁵⁶ - 1) or negative values.

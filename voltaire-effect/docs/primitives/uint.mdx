title: Uint
description: Unsigned integer types for Ethereum (Uint8 to Uint256)
---

Unsigned integers are fundamental to Ethereum—balances, gas, token amounts, and more. voltaire-effect provides [Effect Schema](https://effect.website/docs/schema/introduction/) validation for all fixed-size unsigned integers.

```typescript
import * as Uint from 'voltaire-effect/primitives/Uint'
import * as S from 'effect/Schema'

// Decode from bigint
const value = S.decodeSync(Uint.Uint256)(1000000000000000000n)

// Decode from hex string
const fromHex = S.decodeSync(Uint.Uint256Hex)('0xde0b6b3a7640000')
```

## Available Types

| Type | Bits | Max Value |
|------|------|-----------|
| `Uint8` | 8 | 255 |
| `Uint16` | 16 | 65,535 |
| `Uint32` | 32 | 4,294,967,295 |
| `Uint64` | 64 | 18,446,744,073,709,551,615 |
| `Uint128` | 128 | 2¹²⁸ - 1 |
| `Uint256` | 256 | 2²⁵⁶ - 1 |

## Schemas

### From BigInt

```typescript
import * as Uint from 'voltaire-effect/primitives/Uint'
import * as S from 'effect/Schema'

const gas = S.decodeSync(Uint.Uint64)(21000n)
const balance = S.decodeSync(Uint.Uint256)(1000000000000000000n)
```

### From Number

Safe conversion with overflow checking:

```typescript
const small = S.decodeSync(Uint.Uint8)(255)  // ✓
const overflow = S.decodeSync(Uint.Uint8)(256)  // ✗ ParseError
```

### From Hex String

```typescript
const fromHex = S.decodeSync(Uint.Uint256Hex)('0xde0b6b3a7640000')
// 1000000000000000000n
```

### From Bytes

```typescript
const fromBytes = S.decodeSync(Uint.Uint256Bytes)(
  new Uint8Array([0x0d, 0xe0, 0xb6, 0xb3, 0xa7, 0x64, 0x00, 0x00])
)
```

## U256 Alias

For common 256-bit operations:

```typescript
import * as U256 from 'voltaire-effect/primitives/U256'
import * as S from 'effect/Schema'

const value = S.decodeSync(U256.BigInt)(1000000000000000000n)
const hex = S.encodeSync(U256.Hex)(value)
// "0x0de0b6b3a7640000"
```

## Encoding

Encode back to hex or bytes:

```typescript
const value = S.decodeSync(Uint.Uint256)(1000000000000000000n)

// To hex string
const hex = S.encodeSync(Uint.Uint256Hex)(value)
// "0x0de0b6b3a7640000"

// To bytes
const bytes = S.encodeSync(Uint.Uint256Bytes)(value)
// Uint8Array(32)
```

## Validation

Schemas validate range automatically:

```typescript
import * as Effect from 'effect/Effect'
import * as S from 'effect/Schema'

// Negative values fail
S.decode(Uint.Uint256)(-1n).pipe(
  Effect.catchTag('ParseError', (e) => {
    console.log(e.message)  // "Value must be non-negative"
    return Effect.succeed(0n)
  })
)

// Overflow fails
S.decode(Uint.Uint8)(256n).pipe(
  Effect.catchTag('ParseError', (e) => {
    console.log(e.message)  // "Value exceeds Uint8 max"
    return Effect.succeed(255n)
  })
)
```

## Common Ethereum Values

```typescript
// 1 ETH in wei
const oneEth = S.decodeSync(Uint.Uint256)(1000000000000000000n)

// Gas limit
const gasLimit = S.decodeSync(Uint.Uint64)(21000n)

// ERC-20 amount (USDC has 6 decimals)
const usdcAmount = S.decodeSync(Uint.Uint256)(1000000n)  // 1 USDC

// Block number
const blockNumber = S.decodeSync(Uint.Uint64)(18000000n)
```

## Arithmetic

Use standard bigint operations:

```typescript
const a = S.decodeSync(Uint.Uint256)(100n)
const b = S.decodeSync(Uint.Uint256)(200n)

const sum = a + b       // 300n
const product = a * b   // 20000n
const quotient = b / a  // 2n
```

<Warning>
JavaScript bigint arithmetic can overflow `Uint256`. Re-validate after operations if needed.
</Warning>

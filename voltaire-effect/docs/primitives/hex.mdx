import * as Hex from 'voltaire-effect/primitives/Hex'

## Schema

**Hex.String** — Validates hex strings with or without `0x` prefix.

```typescript
import * as Hex from 'voltaire-effect/primitives/Hex'
import * as S from 'effect/Schema'

const hex = S.decodeSync(Hex.String)('0xdeadbeef')
```

## Functions

**from(input)** — Creates hex from string or Uint8Array.

```typescript
Hex.from('0xdeadbeef')
Hex.from('deadbeef')
Hex.from(new Uint8Array([0xde, 0xad]))
```

**fromBytes(bytes)** — Infallible. Creates hex from bytes.

**toBytes(hex)** — Converts hex to Uint8Array.

**size(hex)** — Returns byte length.

```typescript
const size = await Effect.runPromise(Hex.size('0xdeadbeef'))
// 4
```

## Errors

Catch `InvalidFormatError` on invalid hex input.

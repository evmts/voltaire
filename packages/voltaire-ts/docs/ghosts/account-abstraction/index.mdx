## ERC-4337 Lifecycle

1. **Create** - Build UserOperation with account calldata
2. **Estimate** - Get gas limits from bundler
3. **Sign** - Compute userOpHash and sign
4. **Submit** - Send to bundler mempool
5. **Execute** - Bundler submits to EntryPoint

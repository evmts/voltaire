---
title: Transfer an NFT (ERC721)
description: Transfer an ERC721 NFT to another address using Voltaire's Contract module.
---

## Goal

Transfer an ERC721 NFT from owner to recipient.

Return transaction hash as:

ANSWER: 0x...

## Example (TypeScript)

```ts
import { Contract } from '@voltaire/contract'

const erc721Abi = [
  {
    type: 'function',
    name: 'safeTransferFrom',
    stateMutability: 'nonpayable',
    inputs: [
      { type: 'address', name: 'from' },
      { type: 'address', name: 'to' },
      { type: 'uint256', name: 'tokenId' },
    ],
    outputs: [],
  },
  {
    type: 'function',
    name: 'transferFrom',
    stateMutability: 'nonpayable',
    inputs: [
      { type: 'address', name: 'from' },
      { type: 'address', name: 'to' },
      { type: 'uint256', name: 'tokenId' },
    ],
    outputs: [],
  },
] as const

const bayc = Contract({
  address: '0xBC4CA0EdA7647A8aB7C2061c2E118A18a936f13D',
  abi: erc721Abi,
  provider,
})

const from = '0x742d35Cc6634C0532925a3b844Bc454e4438f44e'
const to = '0x8ba1f109551bD432803012645Ac136ddd64DBA72'
const tokenId = 1234n

// Safe transfer (reverts if recipient can't receive)
const txHash = await bayc.write.safeTransferFrom(from, to, tokenId)
console.log(`ANSWER: 0x${Buffer.from(txHash).toString('hex')}`)
```

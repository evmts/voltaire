---
title: Parameter.encode
description: Encode a single parameter (not implemented)
---

## Overview

`Parameter.encode` is **not implemented** and always throws. Use `Abi.encodeParameters` or `Abi.Function.encodeParams` instead.

## Example

```typescript
import { Abi } from '@tevm/voltaire/Abi';

const params = [
  { type: 'address' },
  { type: 'uint256' }
] as const;

const encoded = Abi.encodeParameters(params, [
  '0x742d35Cc6634C0532925a3b844Bc9e7595f51e3e',
  1000n
]);
```

## Why It Throws

Single-parameter encoding is ambiguous without context. ABI encoding is defined for **parameter lists**, not isolated values.

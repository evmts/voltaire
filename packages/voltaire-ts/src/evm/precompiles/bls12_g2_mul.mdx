# BLS12-381 G2 Mul

BLS12-381 G2 curve scalar multiplication for BLS signature operations.

## Overview

The BLS12-381 G2 Mul precompile provides:
- **G2 Scalar Multiplication**: Multiply G2 point by scalar (k * P)
- **Signature Operations**: Used in BLS signature schemes
- **Fixed Gas**: Constant cost
- **C Library**: Uses blst for performance
- **Address**: 0x000000000000000000000000000000000000000F
- **Since**: Prague (EIP-2537)

**Source**: `src/precompiles/bls12_g2_mul.zig`

## Key Functions

### execute

```zig
const precompiles = @import("precompiles");

pub fn execute(
    allocator: std.mem.Allocator,
    input: []const u8,
    gas_limit: u64,
) PrecompileError!PrecompileResult
```

**Parameters:**
- `allocator`: Memory allocator for result
- `input`: 288 bytes (G2 point + scalar)
- `gas_limit`: Maximum gas available

**Input Format** (288 bytes):
- Bytes 0-255: G2 point (x, y coordinates over F_p²)
- Bytes 256-287: Scalar k (32 bytes, big-endian)

**Returns:** 256 bytes (result point)

**Errors:** `OutOfGas`, `InvalidInput`, `InvalidPoint`

## Gas Costs

**Fixed Cost:** 45000 gas

## Performance

**Native (ReleaseFast)**: ~800-1200 μs
**WASM (ReleaseFast)**: Not supported (uses C library)

# BLS12-381 G1 Mul

BLS12-381 G1 curve scalar multiplication for BLS signature verification.

## Overview

The BLS12-381 G1 Mul precompile provides:
- **G1 Scalar Multiplication**: Multiply G1 point by scalar (k * P)
- **Consensus Support**: Used in Ethereum consensus layer
- **Fixed Gas**: Constant cost
- **C Library**: Uses blst for performance
- **Address**: 0x000000000000000000000000000000000000000C
- **Since**: Prague (EIP-2537)

**Source**: `src/precompiles/bls12_g1_mul.zig`

## Key Functions

### execute

```zig
const precompiles = @import("precompiles");

pub fn execute(
    allocator: std.mem.Allocator,
    input: []const u8,
    gas_limit: u64,
) PrecompileError!PrecompileResult
```

**Parameters:**
- `allocator`: Memory allocator for result
- `input`: 160 bytes (G1 point + scalar)
- `gas_limit`: Maximum gas available

**Input Format** (160 bytes):
- Bytes 0-127: G1 point (x, y coordinates)
- Bytes 128-159: Scalar k (32 bytes, big-endian)

**Returns:** 128 bytes (x, y coordinates of k*P)

**Errors:** `OutOfGas`, `InvalidInput`, `InvalidPoint`

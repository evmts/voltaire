EIP-712 Typed Structured Data Hashing and Signing.

Implementation of EIP-712 for typed structured data hashing and signing.

§Overview

EIP-712 defines a standard for hashing and signing typed structured data, enabling secure off-chain signatures that can be verified on-chain while being human-readable in wallet interfaces.

§Examples

use voltaire::crypto::eip712::{TypedData, Domain, TypeProperty};

// Define the domain
let domain = Domain::new()
    .name("My dApp")
    .version("1")
    .chain_id(1u64);

// Define types
let types = vec![
    ("Message".to_string(), vec![
        TypeProperty::new("content", "string"),
    ]),
];

// Create typed data
let typed_data = TypedData::new(domain, types, "Message")
    .with_string("content", "Hello, EIP-712!");

// Hash for signing
let hash = typed_data.hash();

Struct Address

Source 
pub struct Address(/* private fields */);
Expand description

20-byte Ethereum address.

This type wraps a fixed-size byte array and provides convenient methods for address manipulation, display, and comparison.

§Examples

use voltaire::Address;

// Parse from hex string
let addr: Address = "0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045".parse()?;

// Check if zero address
assert!(!addr.is_zero());

// Display with checksum
println!("{}", addr); // 0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045

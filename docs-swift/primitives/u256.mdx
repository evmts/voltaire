# U256

256-bit unsigned integer. Stored as big-endian bytes.

## Quick Start

```swift
import Voltaire

let one = try U256(hex: "0x01")
print(one.hex) // 0x + 64 hex chars

let raw = [UInt8](repeating: 0xaa, count: 32)
let v = try U256(bytes: raw)
XCTAssertEqual(v.bytes, raw)
```

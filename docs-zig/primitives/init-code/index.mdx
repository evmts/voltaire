## Type Definition (Zig)

```zig
// Represent init code as []const u8; use primitives.Hex to convert hex â‡„ bytes.
```

Init code contains:
1. Constructor logic
2. Constructor arguments (if any)
3. Runtime bytecode (returned by constructor)

## Quick Reference

<Tabs>
<Tab title="Create">

```zig
const Hex = @import("primitives").Hex;
const allocator = std.testing.allocator;
const init_bytes = try Hex.hexToBytes(allocator, "0x6080604052...");
defer allocator.free(init_bytes);
```

</Tab>
<Tab title="Extract">

```zig
// Extract runtime code at known offset: slice init code after constructor
const runtime_bytes = init_bytes[constructor_len..];
// Estimate creation gas: data cost + CREATE cost (documented in EIPs)
```

</Tab>
</Tabs>

## Usage Patterns

### Deploying Contract (JSON-RPC)

```zig
// Build a raw transaction with init code as data (hex)
// {"method":"eth_sendRawTransaction","params":["0xSIGNED_TX"]}
```

### Analyzing Creation Bytecode

```zig
const primitives = @import("primitives");
var bc = try primitives.Bytecode.init(std.testing.allocator, init_bytes);
defer bc.deinit();
// Iterate through instructions until constructor RETURN; then slice to runtime.
```

### Gas Estimation

```zig
// Gas breakdown (reference):
// - 21000 base tx, 32000 CREATE, 16 per non-zero byte, 4 per zero byte
```
